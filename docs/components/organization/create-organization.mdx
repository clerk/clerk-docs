# `<CreateOrganization />`

<Images
  alt="Organization Switcher Example"
  width={992}
  height={568}
  src="/docs/images/ui-components/component-org_switcher.svg"
/>

The `<CreateOrganization />` component is used to render an organization creatio UI that allows users to create brand new organizations within you application.

## Usage

<Tabs items={['Next.js', 'React', 'Remix']}>
  <Tab>
    <CodeBlockTabs options={["App Router", "Pages Router"]}>
      ```jsx copy filename="/app/create-organization/[[...create-organization]]/page.[jsx/tsx]"
      import { CreateOrganization } from "@clerk/nextjs";

      export default function CreateOrganizationPage() {
        return <CreateOrganization />;
      }
      ```
      
      ```jsx copy filename="/pages/create-organization/[[...index]].[jsx/tsx]"
      import { CreateOrganization } from "@clerk/nextjs";

      export default function CreateOrganizationPage() {
        return (
            <CreateOrganization routing="path" path="/create-organization" />
        )
      }
      ```
    </CodeBlockTabs>
  </Tab>

  <Tab>
    ```jsx copy filename="/create-organization.tsx"
    import { CreateOrganization } from "@clerk/clerk-react";

    export default function CreateOrganizationPage() {
      return <CreateOrganization />;
    }
    ```
  </Tab>

  <Tab>
    ```jsx copy filename="/route/create-organization/$.tsx"
    import { CreateOrganization } from "@clerk/remix";

    export default function CreateOrganizationPage() {
      return <CreateOrganization />;
    }
    ```
  </Tab>
</Tabs>
