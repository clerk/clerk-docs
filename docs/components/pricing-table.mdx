The `<PricingTable />` component displays a table of Plans and Features that users can subscribe to.

## Properties

<Properties>
  - `appearance`
  - <code>[Appearance](/docs/customization/overview) | undefined</code>

  Optional object to style your components. Will only affect [Clerk components](/docs/components/overview) and not [Account Portal](/docs/account-portal/overview) pages.

  ---

  - `checkoutProps`
  - `{ appearance: Appearance }`

  Options for the checkout popup. Accepts the following properties:

  - [`appearance`](/docs/customization/overview): an object used to style your components. Will only affect [Clerk components](/docs/components/overview) and not [Account Portal](/docs/account-portal/overview) pages.

  ---

  - `collapseFeatures`
  - `boolean`

  A boolean that indicates whether the features are collapsed. **Requires `layout` to be set to `'default'`**.

  ---

  - `ctaPosition`
  - `'top' | 'bottom'`

  The placement of the CTA button. **Requires `layout` to be set to `'default'`**.

  ---

  - `fallback?`
  - `JSX`

  An optional UI to show when the pricing table is loading.
</Properties>
