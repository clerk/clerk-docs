[Clerk Skills](https://github.com/clerk/skills) are installable packages that give AI coding agents specialized knowledge about Clerk. Once installed, your agent can help you add authentication, manage organizations, sync users, and more.

## Installation

To install all available Skills, use:

```bash
npx skills add clerk/skills
```

To install a specific Skill, use:

```bash
npx skills add clerk/skills --skill <skill-name>
```

Works with Claude Code, Cursor, Windsurf, GitHub Copilot, Codex, Gemini CLI, and [more](https://github.com/nicholasoxford/skills?tab=readme-ov-file#supported-agents).

## Available skills

The following table lists the available Skills. See the linked reference for more information about each Skill.

| Skill | Description |
| - | - |
| [adding-auth](https://github.com/clerk/skills/blob/main/skills/adding-auth/SKILL.md) | Add Clerk to Next.js, Express, Remix, or React |
| [customizing-ui](https://github.com/clerk/skills/blob/main/skills/customizing-ui/SKILL.md) | Build custom sign-in and sign-up forms |
| [syncing-users](https://github.com/clerk/skills/blob/main/skills/syncing-users/SKILL.md) | Sync Clerk users to your database via webhooks |
| [testing-auth](https://github.com/clerk/skills/blob/main/skills/testing-auth/SKILL.md) | E2E auth testing with Playwright or Cypress |
| [managing-orgs](https://github.com/clerk/skills/blob/main/skills/managing-orgs/SKILL.md) | Multi-tenant B2B with Organizations and RBAC |
| [nextjs-patterns](https://github.com/clerk/skills/blob/main/skills/nextjs-patterns/SKILL.md) | Advanced Next.js patterns for auth |
