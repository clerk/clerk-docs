Namespace exposed on `Clerk` for OAuth application / consent helpers.

## Properties

| Property                                                   | Type                                                                                                                        | Description                                                                                                                                                                                                                                      |
| ---------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| <a id="buildconsentactionurl"></a> `buildConsentActionUrl` | <code>(params: \{ clientId: string; \}) => string</code>                                                                    | Returns the URL to use as the `action` attribute of the consent form. Includes `_clerk_session_id` and, in development, the dev browser JWT. Custom-flow developers building their own consent UI use this alongside the `useOAuthConsent` hook. |
| <a id="getconsentinfo"></a> `getConsentInfo`               | <code>(params: GetOAuthConsentInfoParams) => Promise\<[OAuthConsentInfo](/docs/reference/types/oauth-consent-info)\></code> | Loads consent metadata for the given OAuth client for the signed-in user.                                                                                                                                                                        |
