# `<OrganizationSwitcher />`

<Images
  alt="Organization Switcher Example"
  width={992}
  height={568}
  src="/images/ui-components/component-org_switcher.svg"
/>

The `<OrganizationSwitcher />` component is used to enable the ability to switch
between available organizations the user may be part of in your application.

## Usage

<CodeBlockTabs options={['Next.js', 'React', 'Remix']}>
```jsx copy filename="/pages/organization-switcher/[[...index]].tsx"
import { OrganizationSwitcher } from "@clerk/nextjs";

export default function OrganizationSwitcherPage() {
  return (
    <div>
      <OrganizationSwitcher />
    </div>
  );
}
```
```jsx copy filename="organization-switcher.tsx"
import { OrganizationSwitcher } from "@clerk/clerk-react";

export default function OrganizationSwitcherPage() {
  return (
    <div>
      <OrganizationSwitcher />
    </div>
  );
}
```
```jsx copy filename="/pages/organization-switcher/$.tsx"
import { OrganizationSwitcher } from "@clerk/remix";
```
