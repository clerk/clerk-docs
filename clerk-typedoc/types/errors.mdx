Represents the errors that occurred during the last fetch of the parent resource.

## Properties

| Property                     | Type                              | Description                                                   |
| ---------------------------- | --------------------------------- | ------------------------------------------------------------- |
| <a id="fields"></a> `fields` | [`FieldErrors`](field-errors.mdx) | Represents the collection of possible errors on known fields. |
| <a id="global"></a> `global` | <code>unknown[]</code>            | Parsed errors that are not related to any specific field.     |
| <a id="raw"></a> `raw`       | <code>unknown[]</code>            | The raw, unparsed errors from the Clerk API.                  |
