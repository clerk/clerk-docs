| <a id="externalgatewayid"></a> `externalGatewayId`       | `string`              | The identifier for the external payment gateway used for this checkout session.                                     |

| <a id="pathroot"></a> `pathRoot`                         | `string`              | The root path of the resource.                                                                                      |
