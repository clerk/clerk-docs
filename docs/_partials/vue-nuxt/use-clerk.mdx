The following example uses the [`useClerk()`](/docs/reference/composables/use-clerk) composable to access the [`clerk`](/docs/reference/objects/clerk) object. The `clerk` object is used to call the `openSignIn()` method to open the sign-in modal.

```vue
<script setup>
import { useClerk } from '@clerk/vue'

const clerk = useClerk()
</script>

<template>
  <button @click="clerk.openSignIn">Sign in</button>
</template>
```
