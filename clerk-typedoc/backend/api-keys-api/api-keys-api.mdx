## list()

Gets a list of API keys for the given user or Organization. By default, the list is returned in descending order by creation date (newest first).

### Parameters

| Parameter                     | Type                                                    | Description                                                                                                                                                                            |
| ----------------------------- | ------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `queryParams`                 | [`GetAPIKeyListParams`](../get-api-key-list-params.mdx) | -                                                                                                                                                                                      |
| `queryParams.includeInvalid?` | `boolean`                                               | Whether to include invalid API keys (revoked or expired). Defaults to `false`.                                                                                                         |
| `queryParams.limit?`          | `number`                                                | Maximum number of items returned per request. Must be an integer greater than zero and less than `501`. Can be used for paginating the results together with offset. Defaults to `10`. |
| `queryParams.offset?`         | `number`                                                | Skip the first `offset` items when paginating. Needs to be an integer greater or equal to zero. To be used in conjunction with `limit`. Defaults to `0`.                               |
| `queryParams.subject`         | `string`                                                | The user or Organization ID to query API keys by.                                                                                                                                      |

### Returns

`Promise`\<[`PaginatedResourceResponse`](../paginated-resource-response.mdx)\<<code>[APIKey](../api-key.mdx)[]</code>\>\> — A [`PaginatedResourceResponse`](/docs/reference/backend/types/paginated-resource-response) object with a `data` property containing an array of [`APIKey`](/docs/reference/backend/types/backend-api-key) objects and a `totalCount` property containing the total number of API keys for the user or Organization.

## verify()

Verifies the given API key.

- If the API key is valid, the method returns the API key object with its properties.
- If the API key is invalid, revoked, or expired, the method will throw an error.

### Parameters

| Parameter | Type     | Description                          |
| --------- | -------- | ------------------------------------ |
| `secret`  | `string` | The secret of the API key to verify. |

### Returns

`Promise`\<[`APIKey`](../api-key.mdx)\> — The verified [`APIKey`](/docs/reference/backend/types/backend-api-key) object.
