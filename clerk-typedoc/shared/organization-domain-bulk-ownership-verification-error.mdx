A per-domain failure entry returned by the bulk ownership verification flows,
carrying the id of the domain that was skipped and the API error code that
caused it.

## Properties

| Property                 | Type     | Description                                                                                 |
| ------------------------ | -------- | ------------------------------------------------------------------------------------------- |
| <a id="code"></a> `code` | `string` | The API error code describing why the domain was skipped, for example `resource_not_found`. |
| <a id="id"></a> `id`     | `string` | The unique identifier of the Verified Domain that could not be processed.                   |
