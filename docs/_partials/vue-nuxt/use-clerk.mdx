The following example uses the `useClerk()` composable to access the `clerk` object. The `clerk` object is used to call the [`openSignIn()`](/docs/reference/objects/clerk#sign-in) method to open the sign-in modal.

```vue
<script setup>
import { useClerk } from '@clerk/vue'

const clerk = useClerk()
</script>

<template>
  <button @click="clerk.openSignIn">Sign in</button>
</template>
```
