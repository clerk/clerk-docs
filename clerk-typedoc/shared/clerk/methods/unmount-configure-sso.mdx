### `unmountConfigureSSO()`

Unmount a configure SSO component from the target element.
If there is no component mounted at the target node, results in a noop.

```typescript
function unmountConfigureSSO(targetNode: HTMLDivElement): void
```

#### Parameters


| Parameter | Type | Description |
| ------ | ------ | ------ |
| `targetNode` | `HTMLDivElement` | Target node to unmount the ConfigureSSO component from. |
