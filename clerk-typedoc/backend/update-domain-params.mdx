| Name            | Type                         | Description                                                                                                                                                                                                                                                     |
| --------------- | ---------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `domainId`      | `string`                     | The ID of the domain that will be updated.                                                                                                                                                                                                                      |
| `is_secondary?` | <code>boolean \| null</code> | Whether this is a domain for a secondary app, meaning that any subdomain provided is significant and will be stored as part of the domain. This is useful for supporting multiple apps (one primary and multiple secondaries) on the same root domain (eTLD+1). |
