## Backend API (BAPI) endpoint

This method in the SDK is a wrapper around the BAPI endpoint `DELETE/organizations/{organization_id}`. See the [BAPI reference](/docs/reference/backend-api/tag/organizations/DELETE/organizations/\{organization_id}){{ target: '_blank' }} for more information.
