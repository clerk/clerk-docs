## getEnterpriseConnectionList()

Gets the list of enterprise connections for the instance. By default, the list is returned in descending order by creation date (newest first).

### Parameters

| Parameter                | Type                                                                         | Description                                                                                                                                                                                     |
| ------------------------ | ---------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `params`                 | [`EnterpriseConnectionListParams`](../enterprise-connection-list-params.mdx) | -                                                                                                                                                                                               |
| `params.active?`         | `boolean`                                                                    | Filters enterprise connections by active status. If `true`, only active connections are returned. If `false`, only inactive connections are returned. If omitted, all connections are returned. |
| `params.limit?`          | `number`                                                                     | Maximum number of items returned per request. Must be an integer greater than zero and less than `501`. Can be used for paginating the results together with offset. Defaults to `10`.          |
| `params.offset?`         | `number`                                                                     | Skip the first `offset` items when paginating. Needs to be an integer greater or equal to zero. To be used in conjunction with `limit`. Defaults to `0`.                                        |
| `params.organizationId?` | `string`                                                                     | Filters enterprise connections by Organization ID.                                                                                                                                              |

### Returns

`Promise`\<[`PaginatedResourceResponse`](../paginated-resource-response.mdx)\<<code>[EnterpriseConnection](/docs/reference/backend/types/backend-enterprise-connection)[]</code>\>\> — A [`PaginatedResourceResponse`](/docs/reference/backend/types/paginated-resource-response) object with a `data` property containing an array of [`EnterpriseConnection`](/docs/reference/backend/types/backend-enterprise-connection) objects and a `totalCount` property containing the total number of enterprise connections for the instance.
