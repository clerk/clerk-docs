---
title: User Management
description: Manage the signed-in user with the Clerk Android SDK.
sdk: android
---

Use `User` methods to manage the current user's account. You can access the current user reactively with `Clerk.userFlow` or directly with `Clerk.user`.

## Profile

### Update profile

```kotlin
val updatedUser = user.update {
    username = "janedoe"
    firstName = "Jane"
    lastName = "Doe"
}
```

## Email and phone

### Add an email address

```kotlin
val emailAddress = user.createEmailAddress("jane@example.com")
```

### Add a phone number

```kotlin
val phoneNumber = user.createPhoneNumber("+15551234567")
```

### Remove a phone number

```kotlin
phoneNumber.destroy()
```

## External accounts

### Connect with OAuth

```kotlin
val account = user.createExternalAccount(
    provider = OAuthProvider.GOOGLE,
    redirectUrl = "myapp://callback",
    additionalScopes = listOf("email", "profile")
)
```

### Connect with ID token

```kotlin
val account = user.createExternalAccount(
    provider = IdTokenProvider.GOOGLE,
    idToken = idToken
)
```

## Passkeys

### Create a passkey

```kotlin
val passkey = user.createPasskey()
```

## TOTP

### Create a TOTP secret

```kotlin
val totp = user.createTOTP()
```

### Verify a TOTP code

```kotlin
val verifiedTotp = user.verifyTOTP(code = "123456")
```

### Disable TOTP

```kotlin
user.disableTOTP()
```

## Backup codes

### Generate backup codes

```kotlin
val backupCodes = user.createBackupCodes()
```

## Organizations

### List organization invitations

```kotlin
val invitations = user.getOrganizationInvitations()
```

### List organization memberships

```kotlin
val memberships = user.getOrganizationMemberships()
```

### List organization suggestions

```kotlin
val suggestions = user.getOrganizationSuggestions()
```

## Sessions

### List sessions

```kotlin
val sessions = user.getSessions()
```

## Password

### Update password

```kotlin
val updatedUser = user.updatePassword(
    currentPassword = "old-password",
    newPassword = "new-password",
    signOutOfOtherSessions = true
)
```

## Profile image

### Set a profile image

```kotlin
val imageResource = user.setProfileImage(imageData)
```

### Delete a profile image

```kotlin
user.deleteProfileImage()
```

## Delete user

### Delete the current user

```kotlin
user.delete()
```
