---
title: '`EmailAddress`'
description: Create, verify, and delete user email addresses.
sdk: ios
---

`EmailAddress` represents a user's email address and its verification state.

## Create an email address

```swift
let email = try await EmailAddress.create("new@example.com")
```

## Send and verify a code

```swift
let email = try await email.sendCode()
let verified = try await email.verifyCode("123456")
```

## Delete an email address

```swift
_ = try await email.destroy()
```

## Key properties

<Properties>
