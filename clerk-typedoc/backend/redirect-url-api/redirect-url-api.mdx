## createRedirectUrl()

Creates a new redirect URL for the instance.

### Parameters

| Parameter | Type                                                           |
| --------- | -------------------------------------------------------------- |
| `params`  | [`CreateRedirectUrlParams`](../create-redirect-url-params.mdx) |

### Returns

`Promise`\<[`RedirectUrl`](../redirect-url.mdx)\> — The created [`RedirectUrl`](/docs/reference/backend/types/backend-redirect-url) object.

## deleteRedirectUrl()

Deletes the given redirect URL.

### Parameters

| Parameter       | Type     | Description                           |
| --------------- | -------- | ------------------------------------- |
| `redirectUrlId` | `string` | The ID of the redirect URL to delete. |

### Returns

`Promise`\<[`RedirectUrl`](../redirect-url.mdx)\> — The deleted [`RedirectUrl`](/docs/reference/backend/types/backend-redirect-url) object.

## getRedirectUrlList()

Gets a list of whitelisted redirect URLs for the instance. By default, the list is returned in descending order by creation date (newest first).

### Returns

`Promise`\<[`PaginatedResourceResponse`](../paginated-resource-response.mdx)\<<code>[RedirectUrl](../redirect-url.mdx)[]</code>\>\> — A [`PaginatedResourceResponse`](/docs/reference/backend/types/paginated-resource-response) object with a `data` property containing an array of [`RedirectUrl`](/docs/reference/backend/types/backend-redirect-url) objects and a `totalCount` property containing the total number of redirect URLs.
