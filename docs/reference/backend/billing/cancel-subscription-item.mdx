## Backend API (BAPI) endpoint

This method in the SDK is a wrapper around the BAPI endpoint `DELETE /billing/subscription_items/{id}`. See the [BAPI reference](/docs/reference/backend-api/tag/billing/DELETE/billing/subscription_items/%7Bsubscription_item_id%7D){{ target: '_blank' }} for more information.
