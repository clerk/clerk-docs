# `<UserProfile />`

## Overview

The `<UserProfile/>` component is used to render a beautiful, full-featured account management UI that allows users to manage their profile and security settings. Learn more about the `<UserProfile/>` component, what it does, and how it is used by reading our [User Profile component guide](https://clerk.dev/docs/users/user-profile).

## Embedding a `<UserProfile />` component

Using the [Next.js optional catch all route](https://nextjs.org/docs/routing/dynamic-routes#optional-catch-all-routes), you can embed the `<UserProfile/>` component. This allows you to redirect the user inside your application and handle any redirects.

```jsx copy filename="/pages/user-profile/[[...index].[jsx/tsx]"
import { UserProfile } from "@clerk/nextjs";

const UserProfilePage = () => (
  <UserProfile path="/user-profile" routing="path" />
);

export default UserProfilePage;
```

## Custom Styling

The `<UserProfile />` component can be highly customized through the [appearance prop](https://clerk.dev/docs/nextjs/appearance-prop) and can be styled using [CSS Modules](https://clerk.dev/docs/component-customization/using-css-modules), [Tailwind](https://clerk.dev/docs/component-customization/using-tailwind), [inline CSS objects](https://clerk.dev/docs/component-customization/using-inline-css), or [global CSS](https://clerk.dev/docs/component-customization/using-global-css).

## Localization

The `<UserProfile />` component can be localized using the [localization prop](https://clerk.dev/docs/nextjs/localization). This allows you to customize the language used within the `<UserProfile /> `component.
