### `mountConfigureSSO()`

Mount a configure SSO component at the target element.

```typescript
function mountConfigureSSO(targetNode: HTMLDivElement, props?: ConfigureSSOProps): void
```

#### Parameters


| Parameter | Type | Description |
| ------ | ------ | ------ |
| `targetNode` | `HTMLDivElement` | Target to mount the ConfigureSSO component. |
| `props?` | `ConfigureSSOProps` | Configuration parameters. |
