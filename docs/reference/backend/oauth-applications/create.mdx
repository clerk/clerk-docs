<Typedoc src="backend/o-auth-applications-api/methods/create" />

## Usage

<Include src="_partials/backend/usage" />

```tsx
const response = await clerkClient.oauthApplications.create({
  name: 'oauthapp_123',
  redirect_uris: [''],
  scopes: 'profile email public_metadata',
  public: null,
})
```

## Backend API (BAPI) endpoint

This method in the SDK is a wrapper around the BAPI endpoint `POST/oauth_applications`. See the [BAPI reference](/docs/reference/backend-api/tag/oauth-applications/POST/oauth_applications){{ target: '_blank' }} for more information.
