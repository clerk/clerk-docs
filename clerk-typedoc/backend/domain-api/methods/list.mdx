Gets the list of domains for the instance. By default, the list is returned in descending order by creation date (newest first).

Returns a [`PaginatedResourceResponse`](/docs/reference/backend/types/paginated-resource-response) object with a `data` property containing an array of [`Domain`](/docs/reference/backend/types/domain) objects and a `totalCount` property containing the total number of domains for the instance.

```typescript
function list(): Promise<PaginatedResourceResponse<Domain[]>>
```
