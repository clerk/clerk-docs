Holds the ownership verification details of an Organization's domain, including
the TXT record an admin must publish to prove ownership.

## Properties

| Property                                     | Type                                                | Description                                                                                                                                    |
| -------------------------------------------- | --------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------- |
| <a id="attempts"></a> `attempts`             | <code>null \| number</code>                         | The number of verification attempts that have been made, or `null` if none.                                                                    |
| <a id="expiresat"></a> `expiresAt`           | <code>null \| Date</code>                           | The date and time when the current verification attempt expires, or `null`.                                                                    |
| <a id="status"></a> `status`                 | <code>"unverified" \| "verified"</code>             | The current status of the domain ownership verification.                                                                                       |
| <a id="strategy"></a> `strategy`             | <code>"txt" \| "legacy" \| "manual_override"</code> | The strategy used to verify ownership of the domain.                                                                                           |
| <a id="txtrecordname"></a> `txtRecordName`   | <code>null \| string</code>                         | The fully qualified DNS name the org admin must publish the TXT record under. Present only immediately after preparing ownership verification. |
| <a id="txtrecordvalue"></a> `txtRecordValue` | <code>null \| string</code>                         | The exact value the org admin must publish in the TXT record. Present only immediately after preparing ownership verification.                 |
| <a id="verifiedat"></a> `verifiedAt`         | <code>null \| Date</code>                           | The date and time when ownership of the domain was verified, or `null` if it has not been verified.                                            |
