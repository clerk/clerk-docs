The `OrganizationSuggestion` object is the model around [a suggestion to join an Organization](/docs/guides/organizations/add-members/verified-domains#automatic-suggestions).

## Properties

| Property                                                     | Type                                                                                                    | Description                                                                                                               |
| ------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------- |
| <a id="accept"></a> `accept`                                 | <code>() => Promise\<OrganizationSuggestionResource\></code>                                            | Accepts the suggestion, creating a request to join the Organization.                                                      |
| <a id="createdat"></a> `createdAt`                           | `Date`                                                                                                  | The date when the suggestion was created.                                                                                 |
| <a id="id"></a> `id`                                         | `string`                                                                                                | The unique identifier for the suggestion.                                                                                 |
| <a id="publicorganizationdata"></a> `publicOrganizationData` | <code>\{ hasImage: boolean; id: string; imageUrl: string; name: string; slug: null \| string; \}</code> | Public information about the Organization that the suggestion is for.                                                     |
| `publicOrganizationData.hasImage`                            | `boolean`                                                                                               | Whether the Organization has an image.                                                                                    |
| `publicOrganizationData.id`                                  | `string`                                                                                                | The unique identifier for the Organization.                                                                               |
| `publicOrganizationData.imageUrl`                            | `string`                                                                                                | Holds the Organization's logo. Compatible with Clerk's [Image Optimization](/docs/guides/development/image-optimization). |
| `publicOrganizationData.name`                                | `string`                                                                                                | The name of the Organization.                                                                                             |
| `publicOrganizationData.slug`                                | <code>null \| string</code>                                                                             | The URL-friendly identifier of the Organization, or `null` if it has none.                                                |
| <a id="status"></a> `status`                                 | <code>"pending" \| "accepted"</code>                                                                    | The current status of the suggestion.                                                                                     |
| <a id="updatedat"></a> `updatedAt`                           | `Date`                                                                                                  | The date when the suggestion was last updated.                                                                            |
