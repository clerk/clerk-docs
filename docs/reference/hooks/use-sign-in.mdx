The `useSignIn()` hook provides access to the [`SignInFuture`](/docs/reference/objects/sign-in-future) object, which allows you to check the current state of a sign-in attempt and manage the sign-in flow. You can use this to create a [custom sign-in flow](!custom-flow).

<Typedoc src="react/use-sign-in-return" />

> [!IMPORTANT]
> The `SignInFuture` instance referenced by `signIn` does not have a stable identity, and will change as the sign-in flow progresses. Make sure you provide it in dependency arrays when using hooks such as `useEffect`, `useCallback`, or `useMemo`.

## Examples

For example usage of the `useSignIn()` hook, see the [Build your own UI](/docs/guides/development/custom-flows/overview) guides.
