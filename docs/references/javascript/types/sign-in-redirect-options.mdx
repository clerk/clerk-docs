# `SignInRedirectOptions`

An interface that provides options for sign-in redirect methods.

## Properties

| Name | Type | Description |
| --- | --- | --- |
| `afterSignInUrl?` | `string \| null \| undefined` | Full URL or path to navigate after successful sign-in. Defaults to `/`. |
| `afterSignUpUrl?` | `string \| null \| undefined` | Full URL or path to navigate after successful sign-up. Defaults to `/`. |
| `redirectUrl?` | `string \| null \| undefined` | Full URL or path to navigate after successful sign-in, or sign-up. The same as setting `afterSignInUrl` and `afterSignUpUrl` to the same value. |
| `initialValues?` | [`SignInInitialValues`](/docs/references/javascript/types/sign-in-initial-values) | The values used to prefill the sign-in fields with. |
