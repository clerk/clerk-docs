<Typedoc src="backend/organization-api/methods/create-organization-domain" />

## Usage

<Include src="_partials/backend/usage" />

```tsx
const organizationId = 'org_123'
const domainName = 'example.com'
const enrollmentMode = 'automatic_invitation'

const response = await clerkClient.organizations.createOrganizationDomain({
  organizationId,
  domainName,
  enrollmentMode,
})
```

## Backend API (BAPI) endpoint

This method in the SDK is a wrapper around the BAPI endpoint `POST/organizations/{organization_id}/domains`. See the [BAPI reference](/docs/reference/backend-api/tag/organization-domains/POST/organizations/%7Borganization_id%7D/domains){{ target: '_blank' }} for more information.
