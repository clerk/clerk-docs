The current state of the session.

## Properties

| Property                           | Type          | Description                                                                                                                                                                                              |
| ---------------------------------- | ------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| <a id="abandoned"></a> `abandoned` | `"abandoned"` | The session was abandoned client-side.                                                                                                                                                                   |
| <a id="active"></a> `active`       | `"active"`    | The session is valid and all activity is allowed.                                                                                                                                                        |
| <a id="ended"></a> `ended`         | `"ended"`     | The user signed out of the session, but the [`Session`](https://clerk.com/docs/reference/objects/session) remains in the [`Client`](https://clerk.com/docs/reference/objects/client).                    |
| <a id="expired"></a> `expired`     | `"expired"`   | The period of allowed activity for this session has passed.                                                                                                                                              |
| <a id="removed"></a> `removed`     | `"removed"`   | The user signed out of the session and the [`Session`](https://clerk.com/docs/reference/objects/session) was removed from the [`Client`](https://clerk.com/docs/reference/objects/client).               |
| <a id="replaced"></a> `replaced`   | `"replaced"`  | The session has been replaced by another one, but the previous [`Session`](https://clerk.com/docs/reference/objects/session) remains in the [`Client`](https://clerk.com/docs/reference/objects/client). |
| <a id="revoked"></a> `revoked`     | `"revoked"`   | The application ended the session and the [`Session`](https://clerk.com/docs/reference/objects/session) was removed from the [`Client`](https://clerk.com/docs/reference/objects/client).                |
| <a id="pending"></a> `pending`     | `"pending"`   | The user has signed in but hasn't completed [session tasks](https://clerk.com/docs/guides/configure/session-tasks).                                                                                      |
