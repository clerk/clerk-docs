sdk: js-frontend
---

An interface that represents the current [pending task](/docs/authentication/configuration/session-tasks) of a session.

## Properties

<Properties>
  - `key`
  - `'choose-organization'`

  A unique identifier for the task. Currently, the only supported value is `'choose-organization'`, which indicates that the user needs to select an organization before proceeding.
</Properties>
