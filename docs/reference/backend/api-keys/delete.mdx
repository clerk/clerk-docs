```tsx
const response = await clerkClient.apiKeys.delete(apiKeyId)
```

## Backend API (BAPI) endpoint

This method in the SDK is a wrapper around the BAPI endpoint `DELETE/api_keys/{apiKeyID}`. See the [BAPI reference](/docs/reference/backend-api/tag/api-keys/DELETE/api_keys/%7BapiKeyID%7D){{ target: '_blank' }} for more information.
