## Properties

| Property                                  | Type                  | Description                                                                                                                                                                                                                                                       |
| ----------------------------------------- | --------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| <a id="emailaddress"></a> `emailAddress?` | <code>string[]</code> | Counts users with emails that match the given query, via case-insensitive partial match. For example, `emailAddress=hello` will match a user with the email `HELLO@example.com`. Accepts up to 100 email addresses.                                               |
| <a id="externalid"></a> `externalId?`     | <code>string[]</code> | Counts users with the specified external IDs. Accepts up to 100 external IDs.                                                                                                                                                                                     |
| <a id="phonenumber"></a> `phoneNumber?`   | <code>string[]</code> | Counts users with phone numbers that match the given query, via case-insensitive partial match. For example, `phoneNumber=555` will match a user with the phone number `+1555xxxxxxx`. Accepts up to 100 phone numbers.                                           |
| <a id="query"></a> `query?`               | `string`              | Counts users matching the given query across email addresses, phone numbers, usernames, Web3 wallet addresses, user IDs, first names, and last names. Partial matches supported. For example, `query=hello` will match a user with the email `HELLO@example.com`. |
| <a id="userid"></a> `userId?`             | <code>string[]</code> | Counts users with the specified user IDs. Accepts up to 100 user IDs.                                                                                                                                                                                             |
| <a id="username"></a> `username?`         | <code>string[]</code> | Counts users with usernames that match the given query, via case-insensitive partial match. For example, `username=CoolUser` will match a user with the username `SomeCoolUser`. Accepts up to 100 usernames.                                                     |
| <a id="web3wallet"></a> `web3Wallet?`     | <code>string[]</code> | Counts users with Web3 wallet addresses that match the given query, via case-insensitive partial match. For example, `web3Wallet=0x1234567890` will match a user with the Web3 wallet address `0x1234567890`. Accepts up to 100 Web3 wallet addresses.            |
