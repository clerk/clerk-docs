**`Experimental`**

The `CommerceFeatureResource` type represents a feature of a subscription plan.

This is an experimental API for the Billing feature that is available under a public beta, and the API is subject to change. It is advised to pin the SDK version to avoid breaking changes.

## Properties

| Property                               | Type     | Description                                                           | Overrides                                                           |
| -------------------------------------- | -------- | --------------------------------------------------------------------- | ------------------------------------------------------------------- |
| <a id="avatarurl"></a> `avatarUrl`     | `string` | **`Experimental`** The URL of the feature's avatar image.             | -                                                                   |
| <a id="description"></a> `description` | `string` | **`Experimental`** A short description of what the feature provides.  | -                                                                   |
| <a id="id"></a> `id`                   | `string` | **`Experimental`** The unique identifier for the feature.             | [`ClerkResource`](clerk-resource.mdx).[`id`](clerk-resource.mdx#id) |

| <a id="pathroot"></a> `pathRoot`       | `string` | The root path of the resource.                                        | -                                                                   |
| <a id="slug"></a> `slug`               | `string` | **`Experimental`** A unique, URL-friendly identifier for the feature. | -                                                                   |
