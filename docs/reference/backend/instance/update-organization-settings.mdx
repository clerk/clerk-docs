## Usage

<Include src="_partials/backend/usage" />

```ts
const response = await clerkClient.instances.updateOrganizationSettings({
  enabled: true,
  maxAllowedMemberships: 10,
  adminDeleteEnabled: true,
})
```

## Backend API (BAPI) endpoint

This method in the SDK is a wrapper around the BAPI endpoint `PATCH/instance/organization_settings`. See the [BAPI reference](/docs/reference/backend-api/tag/instance-settings/PATCH/instance/organization_settings){{ target: '_blank' }} for more information.
