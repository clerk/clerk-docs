The `OrganizationDomainBulkOwnershipVerificationError` object is a single [Verified Domain](/docs/guides/organizations/add-members/verified-domains) that could not be processed during a bulk ownership verification flow. A failed domain is reported here instead of causing the entire batch to fail.

## Properties

| Property                 | Type     | Description                                                                                 |
| ------------------------ | -------- | ------------------------------------------------------------------------------------------- |
| <a id="code"></a> `code` | `string` | The API error code describing why the domain was skipped, for example `resource_not_found`. |
| <a id="id"></a> `id`     | `string` | The unique identifier of the domain that could not be processed.                            |
