Creates a new session for the given user.

Returns the created [`Session`](/docs/reference/backend/types/backend-session).

```typescript
function createSession(params: CreateSessionParams): Promise<Session>
```

## `CreateSessionParams`


| Property | Type | Description |
| ------ | ------ | ------ |
| <a id="userid"></a> `userId` | `string` | The ID of the user to create a session for. |
