**`Experimental`**

The `CommerceStatementTotals` type represents the total costs, taxes, and other pricing details for a statement.

This is an experimental API for the Billing feature that is available under a public beta, and the API is subject to change. It is advised to pin the SDK version and the clerk-js version to avoid breaking changes.

## Properties

| Property                             | Type                                                                             | Description                                                                                                                                  |
| ------------------------------------ | -------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------- |
| <a id="credit"></a> `credit`         | [`CommerceMoneyAmount`](/docs/references/javascript/types/commerce-money-amount) | **`Experimental`** Any credits (like account balance or promo credits) that are being applied to the checkout.                               |
| <a id="grandtotal"></a> `grandTotal` | [`CommerceMoneyAmount`](/docs/references/javascript/types/commerce-money-amount) | **`Experimental`** The total amount for the checkout, including taxes and after credits/discounts are applied. This is the final amount due. |
| <a id="pastdue"></a> `pastDue`       | [`CommerceMoneyAmount`](/docs/references/javascript/types/commerce-money-amount) | **`Experimental`** Any outstanding amount from previous unpaid invoices that is being collected as part of the checkout.                     |
| <a id="subtotal"></a> `subtotal`     | [`CommerceMoneyAmount`](/docs/references/javascript/types/commerce-money-amount) | **`Experimental`** The price of the items or plan before taxes, credits, or discounts are applied.                                           |
| <a id="taxtotal"></a> `taxTotal`     | [`CommerceMoneyAmount`](/docs/references/javascript/types/commerce-money-amount) | **`Experimental`** The amount of tax included in the checkout.                                                                               |
