## Backend API (BAPI) endpoint

This method in the SDK is a wrapper around the BAPI endpoint `GET/redirect_urls/{id}`. See the [BAPI reference](/docs/reference/backend-api/tag/redirect-urls/GET/redirect_urls){{ target: '_blank' }} for more information.
