The `OrganizationMembershipRequest` object is the model that describes [the request of a user to join an Organization](/docs/guides/organizations/add-members/verified-domains#membership-requests).

## Properties

| Property                                     | Type                                                                | Description                                                                            |
| -------------------------------------------- | ------------------------------------------------------------------- | -------------------------------------------------------------------------------------- |
| <a id="accept"></a> `accept`                 | <code>() => Promise\<OrganizationMembershipRequestResource\></code> | Accepts the Membership Request, adding the user to the Organization.                   |
| <a id="createdat"></a> `createdAt`           | `Date`                                                              | The date when the Membership Request was created.                                      |
| <a id="id"></a> `id`                         | `string`                                                            | The unique identifier for the Membership Request.                                      |
| <a id="organizationid"></a> `organizationId` | `string`                                                            | The ID of the Organization the Membership Request is for.                              |
| <a id="publicuserdata"></a> `publicUserData` | [`PublicUserData`](/docs/reference/types/public-user-data)          | Public information about the user that created the Membership Request.                 |
| <a id="reject"></a> `reject`                 | <code>() => Promise\<OrganizationMembershipRequestResource\></code> | Rejects the Membership Request, declining the user's request to join the Organization. |
| <a id="status"></a> `status`                 | <code>"expired" \| "pending" \| "accepted" \| "revoked"</code>      | The current status of the Membership Request.                                          |
| <a id="updatedat"></a> `updatedAt`           | `Date`                                                              | The date when the Membership Request was last updated.                                 |
