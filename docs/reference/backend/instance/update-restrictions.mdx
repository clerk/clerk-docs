## Usage

<Include src="_partials/backend/usage" />

```ts
const response = await clerkClient.instances.updateRestrictions({
  allowlist: true,
  blockEmailSubaddresses: true,
})
```

## Backend API (BAPI) endpoint

This method in the SDK is a wrapper around the BAPI endpoint `PATCH/instance/restrictions`. See the [BAPI reference](/docs/reference/backend-api/tag/instance-settings/PATCH/instance/restrictions){{ target: '_blank' }} for more information.
