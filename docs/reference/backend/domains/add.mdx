## Example

<Include src="_partials/backend/usage" />

```tsx
const response = await clerkClient.domains.add({
  is_satellite: true,
  name: 'example.com',
})
```

## Backend API (BAPI) endpoint

This method in the SDK is a wrapper around the BAPI endpoint `POST/domains`. See the [BAPI reference](/docs/reference/backend-api/tag/domains/POST/domains){{ target: '_blank' }} for more information.
