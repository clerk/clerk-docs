The following example demonstrates how to use the [`useUser()`](/docs/reference/composables/use-user) composable to access the [`User`](/docs/reference/objects/user) object, which contains the current user's data such as their ID.

```vue {{ filename: 'GetCurrentUser.vue' }}
<script setup>
import { useUser } from '@clerk/vue'

const { isSignedIn, user, isLoaded } = useUser()
</script>

<template>
  <div v-if="!isLoaded">
    <!-- Use `isLoaded` to check if Clerk is loaded -->
    Loading...
  </div>

  <div v-else-if="isSignedIn">Hello {{ user?.id }}!</div>

  <div v-else>Sign in to view this page</div>
</template>
```
