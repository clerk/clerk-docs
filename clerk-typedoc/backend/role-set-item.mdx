The Backend `RoleSetItem` object represents a [Role](/docs/guides/organizations/control-access/roles-and-permissions) that belongs to a [RoleSet](role-set.mdx).

## Properties

| Property                                  | Type                         | Description                                             |
| ----------------------------------------- | ---------------------------- | ------------------------------------------------------- |
| <a id="createdat"></a> `createdAt`        | `number`                     | The Unix timestamp when the Role was first created.     |
| <a id="description"></a> `description`    | <code>null \| string</code>  | A description of the Role.                              |
| <a id="hasmembers"></a> `hasMembers?`     | <code>null \| boolean</code> | Whether any Organization members have this Role.        |
| <a id="id"></a> `id`                      | `string`                     | The unique identifier for the Role.                     |
| <a id="key"></a> `key`                    | `string`                     | The unique key of the Role.                             |
| <a id="memberscount"></a> `membersCount?` | <code>null \| number</code>  | The number of Organization members that have this Role. |
| <a id="name"></a> `name`                  | `string`                     | The name of the Role.                                   |
| <a id="updatedat"></a> `updatedAt`        | `number`                     | The Unix timestamp when the Role was last updated.      |
