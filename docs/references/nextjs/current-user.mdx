# `currentUser()`

The `currentUser` helper returns the [`Backend User`](/docs/references/backend/types/backend-user) object of the currently active user. It can be used in Server Components, Route Handlers, and Server Actions. Under the hood, this helper calls `fetch()` so it is automatically deduped per request.

```tsx filename="app/page.tsx"
import { currentUser } from '@clerk/nextjs/server';

export default async function Page() {
  const user = await currentUser();

  if (!user) return <div>Not signed in</div>;

  return <div>Hello {user?.firstName}</div>;
}
```
