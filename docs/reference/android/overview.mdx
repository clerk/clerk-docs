Use the Clerk Android SDK to handle authentication and user management. Start with the [quickstart guide](/docs/android/getting-started/quickstart) for setup and initialization.

## Auth

See the full [auth reference](/docs/reference/android/auth) for sign-in, sign-up, and session management methods.

## Installation

To install the Clerk Android SDK using Gradle, add the following to your `build.gradle.kts`:

```kotlin
dependencies {
    // For prebuilt UI components (includes API)
    implementation("com.clerk:clerk-android-ui:0.1.0")

    // Or for API only (no UI)
    implementation("com.clerk:clerk-android-api:0.1.0")
}
```
