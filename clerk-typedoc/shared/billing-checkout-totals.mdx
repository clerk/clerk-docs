The `BillingCheckoutTotals` type represents the total costs, taxes, and other pricing details for a checkout session.

## Properties

| Property                                                     | Type                                                                                  | Description                                                                                                                                                                                                                       |
| ------------------------------------------------------------ | ------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| <a id="basefee"></a> `baseFee?`                              | [`BillingMoneyAmount`](/docs/reference/types/billing-money-amount)                    | The base plan fee portion of the totals, before per-unit charges and adjustments.                                                                                                                                                 |
| <a id="credit"></a> `credit`                                 | <code>null \| [BillingMoneyAmount](/docs/reference/types/billing-money-amount)</code> | Any credits (like account balance or promo credits) that are being applied to the checkout.                                                                                                                                       |
| <a id="discounts"></a> `discounts?`                          | <code>null \| [BillingDiscounts](billing-discounts.mdx)</code>                        | Discounts applied to this checkout such as mid-cycle prorated seat discounts.                                                                                                                                                     |
| <a id="grandtotal"></a> `grandTotal`                         | [`BillingMoneyAmount`](/docs/reference/types/billing-money-amount)                    | The total amount for the checkout, including taxes and after credits/discounts are applied. This is the final amount due.                                                                                                         |
| <a id="pastdue"></a> `pastDue`                               | <code>null \| [BillingMoneyAmount](/docs/reference/types/billing-money-amount)</code> | Any outstanding amount from previous unpaid invoices that is being collected as part of the checkout.                                                                                                                             |
| <a id="perunittotals"></a> `perUnitTotals?`                  | <code>[BillingPerUnitTotal](/docs/reference/types/billing-per-unit-total)[]</code>    | Per-unit cost breakdown for items actively being purchased in this checkout (for example, seats being added). When only adding seats mid-cycle, this only covers the seats being added, not seats already paid for.               |
| <a id="subtotal"></a> `subtotal`                             | [`BillingMoneyAmount`](/docs/reference/types/billing-money-amount)                    | The price of items actively being purchased in this checkout, before taxes and discounts. When only adding seats mid-cycle, this reflects just the new seats and excludes the base plan fee and seats that were already paid for. |
| <a id="taxtotal"></a> `taxTotal`                             | [`BillingMoneyAmount`](/docs/reference/types/billing-money-amount)                    | The amount of tax included in the checkout.                                                                                                                                                                                       |
| <a id="totaldueafterfreetrial"></a> `totalDueAfterFreeTrial` | <code>null \| [BillingMoneyAmount](/docs/reference/types/billing-money-amount)</code> | The amount that becomes due after a free trial ends.                                                                                                                                                                              |
| <a id="totalduenow"></a> `totalDueNow`                       | [`BillingMoneyAmount`](/docs/reference/types/billing-money-amount)                    | The amount that needs to be immediately paid to complete the checkout.                                                                                                                                                            |
| <a id="totaldueperperiod"></a> `totalDuePerPeriod`           | [`BillingMoneyAmount`](/docs/reference/types/billing-money-amount)                    | The amount that will be charged per period for this subscription.                                                                                                                                                                 |
| <a id="totalsdueperperiod"></a> `totalsDuePerPeriod?`        | [`BillingPeriodTotals`](billing-period-totals.mdx)                                    | Full renewal period totals after this checkout completes. Contains the complete breakdown of what the next recurring charge will look like, including all seats and the base plan fee.                                            |
