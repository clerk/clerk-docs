{/* JS file: https://github.com/clerk/javascript/blob/main/packages/types/src/appearance.ts#L538 */}

The `captcha` property can be used to change the appearance of the CAPTCHA widget. It is passed as a parameter to the [`appearance` prop](/docs/customization/overview).

## Properties

<Properties>
  - `theme`
  - `'auto' | 'light' | 'dark'`

  The CAPTCHA widget theme. Defaults to `auto`.

  ---

  - `size`
  - `'normal' | 'flexible' | 'compact'`

  The CAPTCHA widget size. Defaults to `normal`.

  ---

  - `language`
  - `string`

  The CAPTCHA widget language. Must be one of the following:

  - `'auto'` (default) to use the language set based on the [localization](/docs/customization/localization) used. Supported languages can be found [here](https://developers.cloudflare.com/turnstile/reference/supported-languages).
  - Language and country code (e.g. `en-US`)
</Properties>
