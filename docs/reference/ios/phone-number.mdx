---
title: '`PhoneNumber`'
description: Create, verify, and manage phone numbers and MFA settings.
sdk: ios
---

`PhoneNumber` represents a user's phone number and its verification state.

## Create a phone number

```swift
let phone = try await PhoneNumber.create("+15555555555")
```

## Send and verify a code

```swift
let phone = try await phone.sendCode()
let verified = try await phone.verifyCode("123456")
```

## Manage MFA flags

```swift
_ = try await phone.setReservedForSecondFactor(reserved: true)
_ = try await phone.makeDefaultSecondFactor()
```

## Delete a phone number

```swift
_ = try await phone.delete()
```

## Key properties

<Properties>
  - `phoneNumber`
  - `String`

  E.164 formatted number.

  ---

  - `reservedForSecondFactor`
  - `Bool`

  Whether it is reserved for 2FA.

  ---

  - `defaultSecondFactor`
  - `Bool`

  Whether it is the default 2FA factor.
</Properties>
