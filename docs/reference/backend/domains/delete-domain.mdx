> [!WARNING]
> This method is deprecated. Use the [`delete()`](/docs/reference/backend/domains/delete) method instead.

Deletes a satellite domain for the instance. It is currently not possible to delete the instance's primary domain. Returns a [`DeletedObject`](/docs/reference/backend/types/deleted-object).

```ts
function deleteDomain(id: string): Promise<DeletedObject>
```

## Parameters

<Properties>
  - `domainId`
  - `string`

  The ID of the domain to delete.
</Properties>

## Example

<Include src="_partials/backend/usage" />

```tsx
const response = await clerkClient.domains.deleteDomain('domain_123')
```
