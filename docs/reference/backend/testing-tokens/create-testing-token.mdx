Creates a [Testing Token](/docs/guides/development/testing/overview#testing-tokens) for the instance.

```ts
function createTestingToken(): Promise<TestingToken>
```

## Example

<Include src="_partials/backend/usage" />

```tsx
const response = await clerk.testingTokens.createTestingToken()
```

## Backend API (BAPI) endpoint

This method in the SDK is a wrapper around the BAPI endpoint `POST/testing_tokens`. See the [BAPI reference](/docs/reference/backend-api/tag/testing-tokens/post/testing_tokens){{ target: '_blank' }} for more information.
