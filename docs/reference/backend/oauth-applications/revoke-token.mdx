---
title: '`revokeToken()`'
description: Use the revokeToken() method to revoke an OAuth application token.
---

> [!IMPORTANT]
> JWT access tokens cannot be revoked. Only [opaque](!opaque-token) OAuth access tokens and refresh tokens can be revoked.

<Typedoc src="backend/o-auth-applications-api/methods/revoke-token" />

## Usage

<Include src="_partials/backend/usage" />

```tsx
const oauthApplicationId = 'oauthapp_123'
const token = 'oat_123'

await clerkClient.oauthApplications.revokeToken({ oauthApplicationId, token })
```

## Backend API (BAPI) endpoint

This method in the SDK is a wrapper around the BAPI endpoint `POST/oauth_applications/{oauth_application_id}/revoke_token`. See the [BAPI reference](/docs/reference/backend-api/tag/oauth-applications/POST/oauth_applications/%7Boauth_application_id%7D/revoke_token){{ target: '_blank' }} for more information.
