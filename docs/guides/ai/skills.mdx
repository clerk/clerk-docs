[Clerk Skills](https://github.com/clerk/skills) are installable packages that give AI coding agents specialized knowledge about Clerk. Once installed, your agent can help you add authentication, manage organizations, sync users, and more.

## Installation

To install all available Skills, use:

```bash
npx skills add clerk/skills
```

To install a specific Skill, use:

```bash
npx skills add clerk/skills --skill {skill-name}
```

Works with Claude Code, Cursor, Windsurf, GitHub Copilot, Codex, Gemini CLI, and more. See [all supported agents](https://github.com/vercel-labs/skills?tab=readme-ov-file#supported-agents).

## Available skills

The following table lists the available Skills. See the linked reference for more information about each Skill.

| Skill | Description | When to use |
| - | - | - |
| [/clerk](https://github.com/clerk/skills/blob/main/skills/clerk/SKILL.md) | Router - routes to the right skill | Always start here |
| [/clerk-setup](https://github.com/clerk/skills/blob/main/skills/setup/SKILL.md) | Add Clerk to any framework | New projects, framework setup |
| [/clerk-custom-ui](https://github.com/clerk/skills/blob/main/skills/custom-ui/SKILL.md) | Custom sign-in/up components and appearance | Building custom forms, styling |
| [/clerk-nextjs-patterns](https://github.com/clerk/skills/blob/main/skills/nextjs-patterns/SKILL.md) | Advanced Next.js patterns | Server Actions, middleware, caching |
| [/clerk-orgs](https://github.com/clerk/skills/blob/main/skills/orgs/SKILL.md) | Multi-tenant B2B organizations | Building team workspaces, RBAC |
| [/clerk-webhooks](https://github.com/clerk/skills/blob/main/skills/webhooks/SKILL.md) | Real-time events and data syncing | Webhooks, database sync, notifications |
| [/clerk-testing](https://github.com/clerk/skills/blob/main/skills/testing/SKILL.md) | E2E testing for auth flows | Writing Playwright/Cypress tests |
