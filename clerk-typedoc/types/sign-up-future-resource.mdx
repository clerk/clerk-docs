The current active `SignUp` instance, for use in custom flows.

| Property                                         | Type                                                                                                                                                                                                                                                                                                                                                                        | Description                                                                                                                                                                            |

Will cause anything observing the session state (such as the `useUser()` hook) to update automatically. |
