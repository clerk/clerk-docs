---
title: '`<UserButton />` (Native)'
description: A native avatar button component that displays the user's profile image and opens a profile management interface.
sdk: expo
---

The native `<UserButton />` component renders a circular avatar showing the user's profile image (or initials as fallback). When tapped, it opens a native profile management modal.

> [!NOTE]
> This documents the native `<UserButton />` from `@clerk/expo/native`. For web projects, see the [web UserButton component](/docs/reference/components/user/user-button).

## Usage

Import from `@clerk/expo/native`:

```tsx
import { UserButton } from '@clerk/expo/native'
```

### Basic example

```tsx {{ filename: 'app/(home)/index.tsx' }}
import { View, Text } from 'react-native'
import { UserButton } from '@clerk/expo/native'
import { useUser } from '@clerk/expo'

export default function HomeScreen() {
  const { user } = useUser()

  return (
    <View style={{ flex: 1, padding: 20 }}>
      <View style={{ flexDirection: 'row', justifyContent: 'space-between', alignItems: 'center' }}>
        <Text style={{ fontSize: 24 }}>Welcome, {user?.firstName}</Text>
        <UserButton />
      </View>
    </View>
  )
}
```

### In a header

```tsx {{ filename: 'app/(home)/_layout.tsx' }}
import { Stack } from 'expo-router'
import { UserButton } from '@clerk/expo/native'

export default function HomeLayout() {
  return (
    <Stack
      screenOptions={{
        headerRight: () => <UserButton style={{ marginRight: 16 }} />,
      }}
    >
      <Stack.Screen name="index" options={{ title: 'Home' }} />
    </Stack>
  )
}
```

### With custom size

```tsx
<UserButton style={{ width: 48, height: 48 }} />
```

### With sign-out callback

```tsx {{ filename: 'components/Header.tsx' }}
import { UserButton } from '@clerk/expo/native'
import { useRouter } from 'expo-router'

export function Header() {
  const router = useRouter()

  return (
    <UserButton
      onSignOut={() => router.replace('/(auth)/sign-in')}
    />
  )
}
```

## Properties

<Properties>
  - `style`
  - `ViewStyle`

  Style object for the button container. Use this to customize size and positioning. Default size is 36x36 pixels.

  ---

  - `onPress`
  - `() => void`

  Optional callback fired when the button is pressed. If not provided, tapping opens the native profile modal.

  ---

  - `onSignOut`
  - `() => void`

  Callback fired when the user signs out through the profile modal. Use this to navigate to your sign-in screen.
</Properties>

## Avatar display

The `<UserButton />` displays the user's avatar in the following priority:

1. **Profile image** - If the user has uploaded a profile image
2. **Initials** - First letters of first and last name
3. **Default** - "U" if no name is available

## What opens when tapped

Tapping the `<UserButton />` opens a native modal with:

- User profile information
- Account management options
- Sign out button

The modal uses the native [`<UserProfileView />`](/docs/reference/expo/user-profile-view) component internally.

## Requirements

- Expo SDK 53 or later
- Development build (not Expo Go)
- `@clerk/expo` plugin in `app.json`
- User must be signed in (use with `<SignedIn>` or check `useUser()`)

```tsx {{ filename: 'app/(home)/index.tsx' }}
import { SignedIn, SignedOut } from '@clerk/expo'
import { UserButton } from '@clerk/expo/native'
import { AuthView } from '@clerk/expo/native'

export default function Screen() {
  return (
    <>
      <SignedIn>
        <UserButton />
      </SignedIn>
      <SignedOut>
        <AuthView />
      </SignedOut>
    </>
  )
}
```
