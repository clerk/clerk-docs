> [!NOTE]
> This documents the native `<UserButton />` from `@clerk/expo/native`. For web projects, see the [web `<UserButton />` component](/docs/reference/components/user/user-button).

The `<UserButton />` component renders a circular avatar that displays the user's profile image, or their initials as fallback. The avatar is shown using the following priority:

1. **Profile image:** Displayed if the user has uploaded one.
1. **Initials:** The first letters of the user's first and last name.
1. **Default:** Displays "U" if no name is available.

When tapped, the `<UserButton />` opens a native profile management modal powered by [`<UserProfileView />`](/docs/reference/expo/native-components/user-profile-view). This modal includes the user's profile information, account management options, and a sign-out button.

## Requirements

<Include src="_partials/expo/expo-requirements-callout" />

In addition to these requirements, this component requires the user to be signed in (use with `<SignedIn>` or check `useUser()`)

```tsx {{ filename: 'app/(home)/index.tsx' }}
import { SignedIn, SignedOut } from '@clerk/expo'
import { UserButton } from '@clerk/expo/native'
import { AuthView } from '@clerk/expo/native'

export default function Screen() {
  return (
    <>
      <SignedIn>
        <UserButton />
      </SignedIn>
      <SignedOut>
        <AuthView />
      </SignedOut>
    </>
  )
}
```

## Usage

The following example shows how to use the `<UserButton />` in your Expo app.

### Basic usage

```tsx {{ filename: 'app/(home)/index.tsx' }}
import { View, Text } from 'react-native'
import { UserButton } from '@clerk/expo/native'
import { useUser } from '@clerk/expo'

export default function HomeScreen() {
  const { user } = useUser()

  return (
    <View style={{ flex: 1, padding: 20 }}>
      <View style={{ flexDirection: 'row', justifyContent: 'space-between', alignItems: 'center' }}>
        <Text style={{ fontSize: 24 }}>Welcome, {user?.firstName}</Text>
        <UserButton />
      </View>
    </View>
  )
}
```

### In a header

```tsx {{ filename: 'app/(home)/_layout.tsx' }}
import { Stack } from 'expo-router'
import { UserButton } from '@clerk/expo/native'

export default function HomeLayout() {
  return (
    <Stack
      screenOptions={{
        headerRight: () => <UserButton style={{ marginRight: 16 }} />,
      }}
    >
      <Stack.Screen name="index" options={{ title: 'Home' }} />
    </Stack>
  )
}
```

### With custom size

```tsx
<UserButton style={{ width: 48, height: 48 }} />
```

### With sign-out callback

```tsx {{ filename: 'components/Header.tsx' }}
import { UserButton } from '@clerk/expo/native'
import { useRouter } from 'expo-router'

export function Header() {
  const router = useRouter()

  return <UserButton onSignOut={() => router.replace('/(auth)/sign-in')} />
}
```

## Properties

<Properties>
  - `style`
  - `ViewStyle`

  Style object for the button container. Use this to customize size and positioning. Default size is 36x36 pixels.

  ---

  - `onPress`
  - `() => void`

  Optional callback fired when the button is pressed. If not provided, tapping opens the native profile modal.

  ---

  - `onSignOut`
  - `() => void`

  Callback fired when the user signs out through the profile modal. Use this to navigate to your sign-in screen.
</Properties>
