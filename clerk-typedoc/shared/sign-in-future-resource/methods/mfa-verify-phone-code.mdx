```typescript
function verifyPhoneCode(params: SignInFutureMFAPhoneCodeVerifyParams): Promise<{ error: null | ClerkError }>
```

#### `SignInFutureMFAPhoneCodeVerifyParams`


| Property | Type | Description |
| ------ | ------ | ------ |
