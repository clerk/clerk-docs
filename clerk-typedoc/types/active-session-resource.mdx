Represents a session resource that has completed all pending tasks
and authentication factors

## Properties

| Property                                                   | Type                                                                                | Description                                                                                                                                             |
| ---------------------------------------------------------- | ----------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------- |
| <a id="end"></a> `end`                                     | <code>() => Promise\<[SessionResource](/docs/reference/javascript/session)\></code> | Marks the session as ended. The session will no longer be active for this `Client` and its status will become **ended**.                                |
| <a id="factorverificationage"></a> `factorVerificationAge` | <code>null \| \[number, number\]</code>                                             | An array where each item represents the number of minutes since the last verification of a first or second factor: `[firstFactorAge, secondFactorAge]`. |
| <a id="id"></a> `id`                                       | `string`                                                                            | The unique identifier for the session.                                                                                                                  |
| <a id="pathroot"></a> `pathRoot`                           | `string`                                                                            | The root path of the resource.                                                                                                                          |
| <a id="status"></a> `status`                               | `"active"`                                                                          | The current state of the session.                                                                                                                       |
| <a id="user"></a> `user`                                   | [`UserResource`](/docs/reference/javascript/user)                                   | The user associated with the session.                                                                                                                   |
