### `mountSignIn()`

Mounts a sign in flow component at the target element.

```typescript
function mountSignIn(targetNode: HTMLDivElement, signInProps?: SignInProps): void
```

#### Parameters


| Parameter | Type | Description |
| ------ | ------ | ------ |
| `targetNode` | `HTMLDivElement` | Target node to mount the SignIn component. |
| `signInProps?` | [`SignInProps`](../sign-in-props.mdx) | sign in configuration parameters. |
