![The OrganizationSwitcherSheet displays Organization management and account switching actions.](/docs/images/ui-components/android-organization-switcher.png){{ style: { maxWidth: '425px' } }}

The `OrganizationSwitcherSheet` renders the same account-switching sheet used by [`OrganizationSwitcher`](/docs/reference/views/organization/organization-switcher), without rendering the switcher trigger. Use it when your app owns the trigger UI and only needs Clerk's prebuilt sheet for managing the [Active Organization](!active-organization), switching accounts, accepting Organization invitations and suggestions, and creating an Organization when allowed.

Render `OrganizationSwitcherSheet` while your trigger's sheet state is open, and remove it from composition from `onDismissRequest`.

## Parameters

<Properties>
  - `onDismissRequest`
  - `() -> Unit`

  Called when the sheet, or any default full-screen follow-up flow, should be dismissed. This parameter is required.

  ---

  - `modifier`
  - `Modifier`

  A modifier that gets applied to `OrganizationSwitcherSheet`.

  ---

  - `clerkTheme`
  - `ClerkTheme?`

  The theme to apply to the `OrganizationSwitcherSheet`.

  ---

  - `showCreateOrganization`
  - `Boolean`

  Whether to show the create-organization row when Organization creation is enabled for the current user. Defaults to `true`.

  ---

  - `onOrganizationChanged`
  - `(() -> Unit)?`

  Called after the user successfully switches to an Organization or [Personal Account](!personal-account).

  ---

  - `hidePersonal`
  - `Boolean`

  Whether to hide the [Personal Account](!personal-account) option when Organization selection is not required. Defaults to `false`.

  ---

  - `onManageOrganization`
  - `((OrganizationMembership) -> Unit)?`

  Called when the user selects the manage Organization action. When `null`, the sheet opens the default [`OrganizationProfileView`](/docs/reference/views/organization/organization-profile-view). When provided, the sheet calls `onDismissRequest` before invoking your callback.

  ---

  - `onCreateOrganization`
  - `((OrganizationCreationDefaults?) -> Unit)?`

  Called when the user selects the create Organization action. When `null`, the sheet opens the default Organization creation flow. When provided, the sheet calls `onDismissRequest` before invoking your callback.

  ---

  - `organizationProfileCustomRows`
  - `List<OrganizationProfileCustomRow>`

  Custom rows forwarded to the default [`OrganizationProfileView`](/docs/reference/views/organization/organization-profile-view) opened by the sheet.

  ---

  - `organizationProfileCustomDestination`
  - `(@Composable (String) -> Unit)?`

  Custom destination builder forwarded to the default [`OrganizationProfileView`](/docs/reference/views/organization/organization-profile-view). The destination receives the tapped row's `routeKey`.
</Properties>

## Usage

The following examples show how to use the `OrganizationSwitcherSheet` in your app.

### Custom trigger

```kotlin
import androidx.compose.material3.Button
import androidx.compose.material3.Text
import androidx.compose.runtime.Composable
import androidx.compose.runtime.getValue
import androidx.compose.runtime.mutableStateOf
import androidx.compose.runtime.saveable.rememberSaveable
import androidx.compose.runtime.setValue
import com.clerk.ui.organizationswitcher.OrganizationSwitcherSheet

@Composable
fun CustomOrganizationSwitcherTrigger() {
    var showSheet by rememberSaveable { mutableStateOf(false) }

    Button(onClick = { showSheet = true }) {
        Text("Switch account")
    }

    if (showSheet) {
        OrganizationSwitcherSheet(
            onDismissRequest = { showSheet = false },
            onOrganizationChanged = {
                // Refresh organization-scoped data.
            },
        )
    }
}
```

### Override default follow-up flows

```kotlin
import com.clerk.api.organizations.OrganizationCreationDefaults
import com.clerk.api.organizations.OrganizationMembership
import com.clerk.ui.organizationswitcher.OrganizationSwitcherSheet

OrganizationSwitcherSheet(
    onDismissRequest = { showSheet = false },
    onManageOrganization = { membership: OrganizationMembership ->
        // Navigate to your own Organization management screen.
    },
    onCreateOrganization = { creationDefaults: OrganizationCreationDefaults? ->
        // Navigate to your own create Organization flow.
    },
)
```

### Hide create Organization

```kotlin
OrganizationSwitcherSheet(
    onDismissRequest = { showSheet = false },
    showCreateOrganization = false,
)
```
