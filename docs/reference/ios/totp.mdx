---
title: '`TOTPResource`'
description: Create and verify TOTP secrets for authenticator apps.
sdk: ios
---

`TOTPResource` represents a TOTP secret used by an authenticator app.

## Create and verify TOTP

```swift
@Environment(Clerk.self) private var clerk

if let user = clerk.user {
  let totp = try await user.createTOTP()
  _ = try await user.verifyTOTP(code: "123456")
  print(totp.uri ?? "")
}
```

## Disable TOTP

```swift
if let user = clerk.user {
  _ = try await user.disableTOTP()
}
```

## Key properties

<Properties>
  - `secret`
  - `String?`

  Returned only on creation.

  ---

  - `uri`
  - `String?`

  TOTP URI for QR or copy/paste.

  ---

  - `verified`
  - `Bool`

  Whether the secret has been verified.

  ---

  - `backupCodes`
  - `[String]?`

  Fresh backup codes when available.
</Properties>
