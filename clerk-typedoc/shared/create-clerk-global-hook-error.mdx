Creates a ClerkGlobalHookError object from a ClerkError instance.
It's a wrapper for all the different instances of Clerk errors that can
be returned when using Clerk hooks.

## Parameters

| Parameter | Type         |
| --------- | ------------ |
| `error`   | `ClerkError` |

## Returns

`ClerkError` & <code>\{ isClerkApiResponseError: \{ (error: unknown): error is ClerkAPIResponseError; (this: unknown): this is ClerkAPIResponseError; \}; isClerkRuntimeError: \{ (error: unknown): error is ClerkRuntimeError; (this: unknown): this is ClerkRuntimeError; \}; \}</code>
