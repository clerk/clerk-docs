Disable all of a user's MFA methods (e.g. [OTP](!otp) sent via SMS, TOTP on their authenticator app) at once.

```typescript
function disableUserMFA(userId: string): Promise<UserID>
```

## Parameters


| Parameter | Type | Description |
| ------ | ------ | ------ |
| `userId` | `string` | The ID of the user to disable MFA for. |
