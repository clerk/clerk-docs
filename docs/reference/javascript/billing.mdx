The `Billing` object provides methods for managing billing for a user or organization.

To use these methods, you must access them through the `Clerk` object: `Clerk.billing`.

> [!NOTE]
> If an `orgId` parameter is not provided, the methods will automatically use the current User ID.

## Methods

### `getPaymentAttempt()`

Returns details of a specific payment attempt for the current user or supplied organization. Returns a `Promise` resolving to a [`BillingPaymentResource`](/docs/reference/javascript/types/billing-payment-resource).

```ts
function getPaymentAttempt(GetPaymentAttemptParams): Promise<BillingPaymentResource>
```

#### `GetPaymentAttemptParams`

<Properties>
  - `id`
  - `string`

  The Payment Attempt ID to fetch.

  ---

  - `orgId?`
  - `string`

  The Organization ID to perform the request on.
</Properties>

### `getPaymentAttempts()`

Returns a list of payment attempts for the current user or supplied organization. Returns a `Promise` resolving to a [`ClerkPaginatedResource`](/docs/reference/javascript/types/clerk-paginated-resource) containing an array of [`BillingPaymentResource`](/docs/reference/javascript/types/billing-payment-resource).

```ts
function getPaymentAttempts(
  params: GetPaymentAttemptsParams,
): Promise<ClerkPaginatedResponse<BillingPaymentResource>>
```

#### `GetPaymentAttemptsParams`

<Properties>
  - `initialPage?`
  - `number`

  A number that specifies which page to fetch. For example, if `initialPage` is set to `10`, it will skip the first 9 pages and fetch the 10th page.

  ---

  - `pageSize?`
  - `number`

  A number that specifies the maximum number of results to return per page.

  ---

  - `orgId?`
  - `string`

  The Organization ID to perform the request on.
</Properties>

### `getPlan()`

```ts
function getPlan(GetPlanParams): Promise<BillingPlanResource>
```

#### `GetPlanParams`

<Properties>
  - `id`
  - `string`

  The Plan ID to fetch.
</Properties>

### `getPlans()`

Returns a list of all publically visible billing plans. Returns a list of payment attempts for the current user or supplied organization. Returns a `Promise` resolving to a [`ClerkPaginatedResource`](/docs/reference/javascript/types/clerk-paginated-resource) containing an array of [`BillingPlanResource`](/docs/reference/javascript/types/billing-plan-resource).

```ts
function getPlans(params?: GetPlansParams): Promise<ClerkPaginatedResponse<BillingPlanResource>>
```

#### `GetPlansParams`

<Properties>
  - `for?`
  - `"user" | "organization"`

  The type of payer for the Plans.

  ---

  - `initialPage?`
  - `number`

  A number that specifies which page to fetch. For example, if `initialPage` is set to `10`, it will skip the first 9 pages and fetch the 10th page.

  ---

  - `pageSize?`
  - `number`

  A number that specifies the maximum number of results to return per page.
</Properties>

### `startCheckout()`

Creates a new billing checkout for the current user or supplied organization. Returns a `Promise` resolving to a [`BillingCheckoutResource`](/docs/reference/javascript/types/billing-checkout-resource).

```ts
function startCheckout(params: CreateCheckoutParams): Promise<BillingCheckoutResource>
```

#### `CreateCheckoutParams`

<Properties>
  - `planId`
  - `string`

  The unique identifier for the Plan.

  ---

  - `planPeriod`
  - `"month" | "annual"`

  The billing period for the Plan.

  ---

  - `orgId?`
  - `string`

  The Organization ID to perform the request on.
</Properties>
