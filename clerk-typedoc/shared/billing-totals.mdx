The `BillingTotals` type represents a granular breakdown of the total amount that will be charged, either during
checkout or at renewal.

## Properties

| Property                                                      | Type                                                                                  | Description                                                |
| ------------------------------------------------------------- | ------------------------------------------------------------------------------------- | ---------------------------------------------------------- |
| <a id="basefee"></a> `baseFee`                                | <code>null \| [BillingMoneyAmount](/docs/reference/types/billing-money-amount)</code> | Base fee component before per-unit charges and adjustments |
| <a id="credit"></a> `credit?`                                 | <code>null \| [BillingMoneyAmount](/docs/reference/types/billing-money-amount)</code> | Credit amount                                              |
| <a id="credits"></a> `credits`                                | <code>null \| BillingCredits</code>                                                   | Unified credits breakdown                                  |
| <a id="discounts"></a> `discounts`                            | <code>null \| [BillingDiscounts](billing-discounts.mdx)</code>                        | Information about the discounts applied to the payment     |
| <a id="grandtotal"></a> `grandTotal`                          | [`BillingMoneyAmount`](/docs/reference/types/billing-money-amount)                    | Grand total amount                                         |
| <a id="pastdue"></a> `pastDue?`                               | <code>null \| [BillingMoneyAmount](/docs/reference/types/billing-money-amount)</code> | Past due amount                                            |
| <a id="perunittotals"></a> `perUnitTotals?`                   | <code>[BillingPerUnitTotal](/docs/reference/types/billing-per-unit-total)[]</code>    | Per-unit total breakdown (for example, seats)              |
| <a id="subtotal"></a> `subtotal`                              | [`BillingMoneyAmount`](/docs/reference/types/billing-money-amount)                    | Subtotal before adjustments                                |
| <a id="taxtotal"></a> `taxTotal`                              | [`BillingMoneyAmount`](/docs/reference/types/billing-money-amount)                    | Total tax amount                                           |
| <a id="totaldueafterfreetrial"></a> `totalDueAfterFreeTrial?` | <code>null \| [BillingMoneyAmount](/docs/reference/types/billing-money-amount)</code> | Total amount due after free trial ends                     |
| <a id="totalduenow"></a> `totalDueNow?`                       | [`BillingMoneyAmount`](/docs/reference/types/billing-money-amount)                    | Total amount due now                                       |
| <a id="totaldueperperiod"></a> `totalDuePerPeriod?`           | [`BillingMoneyAmount`](/docs/reference/types/billing-money-amount)                    | The expected total payment for each future billing period  |
| <a id="totalsdueperperiod"></a> `totalsDuePerPeriod?`         | [`BillingPeriodTotals`](billing-period-totals.mdx)                                    | Per-period renewal totals, broken down granularly          |
