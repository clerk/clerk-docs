## Example

<Include src="_partials/backend/usage" />

```tsx
const response = await clerkClient.domains.update({
  domainId: 'domain_123',
  is_secondary: true,
})
```

## Backend API (BAPI) endpoint

This method in the SDK is a wrapper around the BAPI endpoint `PATCH/domains/{domain_id}`. See the [BAPI reference](/docs/reference/backend-api/tag/domains/PATCH/domains/\{domain_id}){{ target: '_blank' }} for more information.
