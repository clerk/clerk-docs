The `LastAuthenticationStrategy` enum is used to indicate the last authentication strategy used by a client.

```tsx
type LastAuthenticationStrategy =
  | 'email_address'
  | 'email_code'
  | 'email_link'
  | 'phone_code'
  | 'password'
  | 'username'
  | [`OAuthStrategy`](/docs/reference/javascript/types/sso)
  | [`Web3Strategy`](/docs/reference/javascript/types/web3)
```
