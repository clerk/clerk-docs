The `BillingSubscriptionItemSeats` type represents seat entitlements attached to a subscription item.

## Properties

| Property                         | Type                        | Description                                                                                  |
| -------------------------------- | --------------------------- | -------------------------------------------------------------------------------------------- |
| <a id="quantity"></a> `quantity` | <code>null \| number</code> | The seat limit active while the parent subscription item was active. `null` means unlimited. |
