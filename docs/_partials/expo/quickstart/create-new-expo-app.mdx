If you don't already have an Expo app, run the following commands to [create a new one](https://docs.expo.dev/tutorial/create-your-first-app/).

```npm
npx create-expo-app@latest clerk-expo
cd clerk-expo
```

The default template includes `react-native-reanimated`, which has a [known Android build issue](https://github.com/software-mansion/react-native-reanimated/issues). If you don't need it, remove it to avoid build errors:

```npm
npm uninstall react-native-reanimated react-native-worklets --legacy-peer-deps
```

Then remove `import 'react-native-reanimated'` from `app/_layout.tsx`.
