### `createEmailAddress()`

Adds an email address for the user. A new [`EmailAddress`](/docs/reference/types/email-address) will be created and associated with the user.

> [!WARNING]
> [**Email** must be enabled](/docs/guides/configure/auth-strategies/sign-up-sign-in-options#email) in your app's settings in the Clerk Dashboard.

```typescript
function createEmailAddress(params: CreateEmailAddressParams): Promise<EmailAddressResource>
```

#### `CreateEmailAddressParams`


| Property | Type | Description |
| ------ | ------ | ------ |
| <a id="email"></a> `email` | `string` | The email address to add to the user. |
