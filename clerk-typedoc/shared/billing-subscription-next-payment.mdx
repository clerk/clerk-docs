The `BillingSubscriptionNextPayment` type represents the upcoming payment details for a subscription.

## Properties

| Property                                    | Type                                                                               | Description                                                 |
| ------------------------------------------- | ---------------------------------------------------------------------------------- | ----------------------------------------------------------- |
| <a id="amount"></a> `amount`                | [`BillingMoneyAmount`](/docs/reference/types/billing-money-amount)                 | The amount of the next payment.                             |
| <a id="date"></a> `date`                    | `Date`                                                                             | The date when the next payment is due.                      |
| <a id="perunittotals"></a> `perUnitTotals?` | <code>[BillingPerUnitTotal](/docs/reference/types/billing-per-unit-total)[]</code> | Per-unit cost breakdown for the next payment (e.g., seats). |
| <a id="totals"></a> `totals?`               | [`BillingTotals`](billing-totals.mdx)                                              | Full cost breakdown for the next payment.                   |
