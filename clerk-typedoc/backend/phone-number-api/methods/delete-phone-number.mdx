Deletes the given phone number.

Returns the [`DeletedObject`](/docs/reference/backend/types/deleted-object) object.

```typescript
function deletePhoneNumber(phoneNumberId: string): Promise<DeletedObject>
```

## Parameters


| Parameter | Type | Description |
| ------ | ------ | ------ |
| `phoneNumberId` | `string` | The ID of the phone number to delete. |
