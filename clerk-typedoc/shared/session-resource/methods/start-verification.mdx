### `startVerification()`

Initiates the reverification flow.

Returns a [`SessionVerification`](/docs/reference/types/session-verification) instance with its status and supported factors.

```typescript
function startVerification(params: SessionVerifyCreateParams): Promise<SessionVerificationResource>
```

#### `SessionVerifyCreateParams`


| Property | Type | Description |
| ------ | ------ | ------ |
| <a id="level"></a> `level` | <code>"first_factor" \| "second_factor" \| "multi_factor"</code> | The level of the verification to create. |
