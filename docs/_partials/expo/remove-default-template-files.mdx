The default Expo template includes files that will conflict with the routes you'll create in this quickstart. Remove them before continuing:

```bash
rm -rf "app/(tabs)" app/modal.tsx app/+not-found.tsx
```

> [!IMPORTANT]
> You can skip this step if you used `npx create-expo-app@latest --template blank` to create your app. However, the blank template doesn't include [Expo Router](https://docs.expo.dev/router/introduction/) or themed components. You'll need to install `expo-router` and its dependencies to follow along with this quickstart.
