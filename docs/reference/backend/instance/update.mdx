## Backend API (BAPI) endpoint

This method in the SDK is a wrapper around the BAPI endpoint `PATCH/instance`. See the [BAPI reference](/docs/reference/backend-api/tag/instance-settings/PATCH/instance){{ target: '_blank' }} for more information.
