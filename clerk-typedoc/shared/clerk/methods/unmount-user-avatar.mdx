### `unmountUserAvatar()`

Unmount a user avatar component at the target element.
If there is no component mounted at the target node, results in a noop.

```typescript
function unmountUserAvatar(targetNode: HTMLDivElement): void
```

#### Parameters


| Parameter | Type | Description |
| ------ | ------ | ------ |
| `targetNode` | `HTMLDivElement` | Target node to unmount the UserAvatar component from. |
