Gets a list of whitelisted redirect URLs for the instance. By default, the list is returned in descending order by creation date (newest first).

Returns a [`PaginatedResourceResponse`](/docs/reference/backend/types/paginated-resource-response) object with a `data` property containing an array of [`RedirectUrl`](/docs/reference/backend/types/backend-redirect-url) objects and a `totalCount` property containing the total number of redirect URLs.

```typescript
function getRedirectUrlList(): Promise<PaginatedResourceResponse<RedirectUrl[]>>
```
