# `createOrganization()`

The `createOrganization` method is used to create an organization programatically.


## Usage

<CodeBlockTabs options={['NPM Module', 'window.Clerk']}>
```typescript {8}
import Clerk from '@clerk/clerk-js';

const clerk = new Clerk('pk_[publishable_key]');
await clerk.load();

// ...

clerk.createOrganization({name: "Test", slug: "test"})
```
```html {12}
<script>
const script = document.createElement('script');
script.setAttribute('data-clerk-publishable-key', 'pk_[publishable_key]');
script.async = true;
script.src = `https://[your-domain].clerk.accounts.dev/npm/@clerk/clerk-js@latest/dist/clerk.browser.js`;

script.addEventListener('load', async function () {
    await window.Clerk.load();

    // ...

    clerk.createOrganization({name: "Test", slug: "test"})
});
document.body.appendChild(script);
</script>
```
</CodeBlockTabs>

## Errors

There are a few errors you might run into when trying to use this method.

### 401 Auth Error

If this method fails with a 401 request and the message:

```json
{ "message": "Signed out", "long_message": "You are signed out", "code": "signed_out" }
```

Make sure your user is authenticated before attempting to run the `createOrganization` method.

### 403 Forbidden

Make sure you enable organizations in your Dashboard before using this method. If this feature it not enabled, you'll end up with the following error message:

```json
{ "message": "access denied", "long_message": "The organizations feature is not enabled for this instance. You can enable it at https://dashboard.clerk.com.", "code": "organization_not_enabled_in_instance" }
```
