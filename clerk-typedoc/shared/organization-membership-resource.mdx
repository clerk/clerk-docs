The `OrganizationMembership` object is the model around a user's membership in an Organization.

## Properties

| Property                                      | Type                                                                                                                                                      | Description                                                                                                                                                                                                                |
| --------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| <a id="createdat"></a> `createdAt`            | `Date`                                                                                                                                                    | The date when the membership was created.                                                                                                                                                                                  |
| <a id="destroy"></a> `destroy`                | <code>() => Promise\<OrganizationMembershipResource\></code>                                                                                              | Deletes the membership, removing the user from the Organization.                                                                                                                                                           |
| <a id="id"></a> `id`                          | `string`                                                                                                                                                  | The unique identifier for the membership.                                                                                                                                                                                  |
| <a id="organization"></a> `organization`      | [`OrganizationResource`](/docs/reference/objects/organization)                                                                                            | The [`Organization`](/docs/reference/types/organization) object the membership belongs to.                                                                                                                                 |
| <a id="permissions"></a> `permissions`        | <code>Autocomplete\<OrganizationSystemPermissionKey\>[]</code>                                                                                            | The [Permissions](/docs/guides/organizations/control-access/roles-and-permissions) the member has in the Organization.                                                                                                     |
| <a id="publicmetadata"></a> `publicMetadata`  | [OrganizationMembershipPublicMetadata](/docs/reference/types/metadata#organization-membership-public-metadata)                                            | Metadata that can be read from both the [Frontend API](/docs/reference/frontend-api){{ target: '_blank' }} and [Backend API](/docs/reference/backend-api){{ target: '_blank' }}, but can be set only from the Backend API. |
| <a id="publicuserdata"></a> `publicUserData?` | [`PublicUserData`](/docs/reference/types/public-user-data)                                                                                                | Public information about the user that this membership belongs to.                                                                                                                                                         |
| <a id="role"></a> `role`                      | `string`                                                                                                                                                  | The [Role](/docs/guides/organizations/control-access/roles-and-permissions) of the member in the Organization.                                                                                                             |
| <a id="rolename"></a> `roleName`              | `string`                                                                                                                                                  | The name of the [Role](/docs/guides/organizations/control-access/roles-and-permissions) of the member in the Organization.                                                                                                 |
| <a id="update"></a> `update`                  | <code>(updateParams: [UpdateOrganizationMembershipParams](update-organization-membership-params.mdx)) => Promise\<OrganizationMembershipResource\></code> | Updates the member's [Role](/docs/guides/organizations/control-access/roles-and-permissions) in the Organization.                                                                                                          |
| <a id="updatedat"></a> `updatedAt`            | `Date`                                                                                                                                                    | The date when the membership was last updated.                                                                                                                                                                             |
