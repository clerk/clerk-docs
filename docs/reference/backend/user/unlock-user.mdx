## Backend API (BAPI) endpoint

This method in the SDK is a wrapper around the BAPI endpoint `POST/users/{user_id}/unlock`. See the [BAPI reference](/docs/reference/backend-api/tag/users/POST/users/\{user_id}/unlock){{ target: '_blank' }} for more information.
