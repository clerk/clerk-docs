### `prepareFirstFactorVerification()`

Initiates the [first factor verification](!first-factor-verification) process. This is a required step to complete a reverification flow when using a preparable factor.

Returns a [`SessionVerification`](/docs/reference/types/session-verification) instance with its status and supported factors.

```typescript
function prepareFirstFactorVerification(factor: SessionVerifyPrepareFirstFactorParams): Promise<SessionVerificationResource>
```
