# Accessing the API

The Clerk Next.js and Node.js SDK's mirror the [Backend API](https://github.com/clerkinc/javascript/tree/main/packages/backend/src/api/endpoints). Provided is a list of the available Clerk instance methods.

<Callout>
  The backend API is accessible through the `clerkClient` object if you are using Next.js, or the `clerk` object if you are using Node.js.
</Callout>

## User operations

User operations are exposed by the users sub-api `(clerkClient.users)`. The following methods are available:

### Get

- `getUserList()`
- `getUser()`
- `getCount()`

### Create

- `createUser()`

### Update

- `updateUser()`

### Delete

- `deleteUser()`

## Organization operations

Organization operations are exposed by the organizations sub-api `(clerkClient.organization)`. The following methods are available:

### Get

- `getOrganization()`
- `getOrganizationList()`
- `getOrganizationMembershipList()`
- `getPendingOrganizationInvitationList()`

### Create

- `createOrganization()`
- `createOrganizationMembership()`
- `createOrganizationInvitation()`

### Update

- `updateOrganization()`
- `updateOrganizationMembership()`
- `updateOrganizationMetadata()`
- `updateOrganizationMembershipMetadata()`

### Delete

- `deleteOrganization()`
- `deleteOrganizationMembership()`
- `revokeOrganizationInvitation()`
