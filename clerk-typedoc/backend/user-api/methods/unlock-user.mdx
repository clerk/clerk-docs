Removes a sign-in lock from the given [`User`](/docs/reference/backend/types/backend-user), allowing them to sign in again. See the [guide on user locks](/docs/guides/secure/user-lockout).

```typescript
function unlockUser(userId: string): Promise<User>
```

## Parameters


| Parameter | Type | Description |
| ------ | ------ | ------ |
| `userId` | `string` | The ID of the user to unlock. |
