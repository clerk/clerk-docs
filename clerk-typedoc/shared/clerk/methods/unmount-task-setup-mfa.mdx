### `unmountTaskSetupMFA()`

Unmount a TaskSetupMFA component from the target element.
If there is no component mounted at the target node, results in a noop.

```typescript
function unmountTaskSetupMFA(targetNode: HTMLDivElement): void
```

#### Parameters


| Parameter | Type | Description |
| ------ | ------ | ------ |
| `targetNode` | `HTMLDivElement` | Target node to unmount the TaskSetupMFA component from. |
