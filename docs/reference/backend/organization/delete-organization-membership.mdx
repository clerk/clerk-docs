## Usage

<Include src="_partials/backend/usage" />

```tsx
const organizationId = 'org_123'
const userId = 'user_123'

const response = await clerkClient.organizations.deleteOrganizationMembership({
  organizationId,
  userId,
})
```

## Backend API (BAPI) endpoint

This method in the SDK is a wrapper around the BAPI endpoint `DELETE/organizations/{organization_id}/memberships/{user_id}`. See the [BAPI reference](/docs/reference/backend-api/tag/organization-memberships/DELETE/organizations/\{organization_id}/memberships/\{user_id}){{ target: '_blank' }} for more information.
