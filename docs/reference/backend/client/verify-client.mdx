title: '`verifyClient()`'

## Usage

<Include src="_partials/backend/usage" />

```tsx
const token = 'my-session-token'

const response = await clerkClient.clients.verifyClient(token)
```

## Backend API (BAPI) endpoint

This method in the SDK is a wrapper around the BAPI endpoint `POST/clients/verify`. See the [BAPI reference](/docs/reference/backend-api/tag/clients/POST/clients/verify){{ target: '_blank' }} for more information.
