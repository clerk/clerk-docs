## Properties

| Property                  | Type                                      | Description                                                                                                                                                                                                                                           |
| ------------------------- | ----------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| <a id="flow"></a> `flow?` | <code>"autofill" \| "discoverable"</code> | The flow to use for the passkey sign-in. <ul> <li>`'autofill'`: The client prompts your users to select a passkey before they interact with your app.</li> <li>`'discoverable'`: The client requires the user to interact with the client.</li> </ul> |
