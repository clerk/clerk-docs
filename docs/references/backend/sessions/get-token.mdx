# `getToken()`

Retrieves a token for a JWT Template that is defined in the Clerk Dashboard on the **[JWT Templates](https://dashboard.clerk.com/last-active?path=jwt-templates)** page.

```tsx
function getToken: (sessionId: string, template: string) => Promise<ClerkBackendApiResponse<Token>>;
```

## `getToken()` Parameters

| Name | Type | Description |
| --- | --- | --- |
| `sessionId` | `string` | The ID of the session to retrieve a token for. |
| `template` | `string` | The name of the JWT template from the [Clerk Dashboard](https://dashboard.clerk.com/last-active?path=jwt-templates) to generate a new token from.<br/>E.g. 'firebase', 'grafbase', or your custom template's name. |

## `getToken()` Example

```tsx
const sessionId = 'sess_2b6M8I4VZ1wMtgXGQeDe3SJD3cG';

const template = 'test';
 
const response = await clerkClient.sessions.getToken(sessionId, template);

console.log(response);
/*
{
  data: _Token {
    jwt: 'eyJhbG...'
  },
  errors: null
}
*/
```
