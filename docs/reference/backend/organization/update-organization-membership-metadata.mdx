## Usage

<Include src="_partials/backend/usage" />

```tsx
const organizationId = 'org_123'
const userId = 'user_123'

const response = await clerkClient.organizations.updateOrganizationMembershipMetadata({
  organizationId,
  userId,
  publicMetadata: {
    example: 'this value is updated!',
  },
})
```

## Backend API (BAPI) endpoint

This method in the SDK is a wrapper around the BAPI endpoint `PATCH/organizations/{organization_id}/memberships/{user_id}/metadata`. See the [BAPI reference](/docs/reference/backend-api/tag/organization-memberships/PATCH/organizations/\{organization_id}/memberships/\{user_id}/metadata){{ target: '_blank' }} for more information.
