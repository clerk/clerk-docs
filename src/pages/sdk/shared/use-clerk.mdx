# useClerk

The `useClerk` hook provides access to the the Clerk object, giving you the ability to build alternatives to any Clerk Component.

<Callout type="warning">
This is intended to be used for advanced use cases, like building a completely custom oauth flow or as an escape hatch giving access to lower-level APIs.
</Callout>

## Usage 

<Tabs items={['Next.js', 'React', 'Remix', 'Gatsby']}>
<Tab>
```tsx copy filename="pages/index.tsx"
import { useClerk } from "@clerk/nextjs";

export default function Home() {
  const clerk = useClerk();

  return <button onClick={() => clerk.openSignIn({})}>Sign in</button>;
}
```
