### `mountOAuthConsent()`

Mounts a OAuth consent component at the target element.

```typescript
function mountOAuthConsent(targetNode: HTMLDivElement, oauthConsentProps?: OAuthConsentProps): void
```

#### Parameters


| Parameter | Type | Description |
| ------ | ------ | ------ |
| `targetNode` | `HTMLDivElement` | Target node to mount the OAuth consent component. |
| `oauthConsentProps?` | `OAuthConsentProps` | OAuth consent configuration parameters. |
