## Roles

- **Owner** - Full access to everything including organization member management and instance keys
- **Admin** - Manage applications, instances, billing, config, API keys, and instance keys. Can impersonate users. Cannot manage org members.
- **Developer** - Manage configuration and restrictions. View API keys and billing. Dev-only impersonation.
- **Support** - Read-only config. Manage restrictions and impersonate users for troubleshooting.
- **Member** - Read-only access to configuration. Lowest privileged role for basic dashboard visibility.
