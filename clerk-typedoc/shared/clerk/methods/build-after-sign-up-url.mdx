### `buildAfterSignUpUrl()`

Returns the configured `afterSignUpUrl` of the instance.

```typescript
function buildAfterSignUpUrl(params?: { params?: URLSearchParams }): string
```

#### Parameters


| Parameter | Type | Description |
| ------ | ------ | ------ |
| `params?` | <code>\{ params?: URLSearchParams; \}</code> | Optional query parameters to append to the URL. |
