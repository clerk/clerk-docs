# Accessing the API

Placeholder text: By leveraging the Clerk Backend API, you can easily access and update data...

<Callout>
  The backend API is accessible through the `clerkClient` object if you are using Next.js, or the `clerk` object if you are using Node.js.
</Callout>

## User Object

User operations are exposed by the users sub-api `(clerkClient.users)`. The following methods are available:

- `getUserList()`
- `getUser()`
- `getCount()`
- `createUser()`
- `updateUser()`
- `deleteUser()`

## Organization Object

### Create

- `createOrganization()`
- `createOrganizationMembership()`
- `createOrganizationInvitation()`

### Get

- `getOrganization()`
- `getOrganizationList()`
- `getOrganizationMembershipList()`
- `getPendingOrganizationInvitationList()`

### Update

- `updateOrganization()`
- `updateOrganizationMembership()`
- `updateOrganizationMetadata()`
- `updateOrganizationMembershipMetadata()`

### Delete

- `deleteOrganization()`
- `deleteOrganizationMembership()`
- `revokeOrganizationInvitation()`
