Generates a Clerk JS script URL.

## Parameters

| Parameter | Type                       | Description                                               |
| --------- | -------------------------- | --------------------------------------------------------- |
| `opts`    | `LoadClerkJsScriptOptions` | The options to use when building the Clerk JS script URL. |

## Returns

`string`

## Example

```ts
clerkJsScriptUrl({ publishableKey: "pk_" });
```
