Gets the list of allowlist identifiers for the instance. By default, the list is returned in descending order by creation date (newest first).

Returns a [`PaginatedResourceResponse`](/docs/reference/backend/types/paginated-resource-response) object with a `data` property containing an array of [`AllowlistIdentifier`](/docs/reference/backend/types/backend-allowlist-identifier) objects and a `totalCount` property containing the total number of allowlist identifiers for the instance.

```typescript
function getAllowlistIdentifierList(params: ClerkPaginationRequest): Promise<PaginatedResourceResponse<AllowlistIdentifier[]>>
```

## `ClerkPaginationRequest`


| Property | Type | Description |
| ------ | ------ | ------ |
| <a id="limit"></a> `limit?` | `number` | Maximum number of items returned per request. Must be an integer greater than zero and less than `501`. Can be used for paginating the results together with offset. Defaults to `10`. |
| <a id="offset"></a> `offset?` | `number` | Skip the first `offset` items when paginating. Needs to be an integer greater or equal to zero. To be used in conjunction with `limit`. Defaults to `0`. |
