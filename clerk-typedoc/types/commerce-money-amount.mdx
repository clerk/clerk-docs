**`Experimental`**

The `CommerceMoneyAmount` type represents a monetary value with currency information.

This is an experimental API for the Billing feature that is available under a public beta, and the API is subject to change. It is advised to pin the SDK version and the clerk-js version to avoid breaking changes.

## Properties

| Property                                       | Type     | Description                                                                                                                                       |
| ---------------------------------------------- | -------- | ------------------------------------------------------------------------------------------------------------------------------------------------- |
| <a id="amount"></a> `amount`                   | `number` | **`Experimental`** The raw amount as a number, usually in the smallest unit of the currency (like cents for USD). For example, `1000` for $10.00. |
| <a id="amountformatted"></a> `amountFormatted` | `string` | **`Experimental`** The amount as a formatted string. For example, `10.00` for $10.00.                                                             |
| <a id="currency"></a> `currency`               | `string` | **`Experimental`** The ISO currency code for this amount. For example, `USD`.                                                                     |
| <a id="currencysymbol"></a> `currencySymbol`   | `string` | **`Experimental`** The symbol for the currency. For example, `$`.                                                                                 |
