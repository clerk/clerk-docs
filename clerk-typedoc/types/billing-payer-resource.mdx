The `BillingPayerResource` type represents a payer associated with a billing subscription.

## Properties

| Property                                          | Type                        | Description                                                           |
| ------------------------------------------------- | --------------------------- | --------------------------------------------------------------------- |
| <a id="createdat"></a> `createdAt`                | `Date`                      | The date and time when the payer was created.                         |
| <a id="email"></a> `email?`                       | `string`                    | The email address of the payer.                                       |
| <a id="firstname"></a> `firstName?`               | `string`                    | The first name of the payer.                                          |
| <a id="id"></a> `id`                              | `string`                    | The unique identifier for the payer.                                  |
| <a id="imageurl"></a> `imageUrl`                  | <code>null \| string</code> | The URL of the payer's avatar image.                                  |
| <a id="lastname"></a> `lastName?`                 | `string`                    | The last name of the payer.                                           |
| <a id="organizationid"></a> `organizationId?`     | `string`                    | The unique identifier for the organization that the payer belongs to. |
| <a id="organizationname"></a> `organizationName?` | `string`                    | The name of the organization that the payer belongs to.               |
| <a id="pathroot"></a> `pathRoot`                  | `string`                    | The root path of the resource.                                        |
| <a id="updatedat"></a> `updatedAt`                | `Date`                      | The date and time when the payer was last updated.                    |
| <a id="userid"></a> `userId?`                     | `string`                    | The unique identifier for the payer.                                  |
