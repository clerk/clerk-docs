Updates the given API key.

Returns the updated [`APIKey`](/docs/reference/backend/types/backend-api-key) object.

```typescript
function update(params: UpdateAPIKeyParams): Promise<APIKey>
```

## `UpdateAPIKeyParams`


| Property | Type | Description |
| ------ | ------ | ------ |
| <a id="apikeyid"></a> `apiKeyId` | `string` | The ID of the API key to update. |
| <a id="claims"></a> `claims?` | <code>Record\<string, any\> \| null</code> | Custom claims to store additional information about the API key. |
| <a id="description"></a> `description?` | <code>string \| null</code> | The description of the API key. |
| <a id="scopes"></a> `scopes?` | <code>string[]</code> | Scopes to limit the API key's access to specific resources. |
| <a id="secondsuntilexpiration"></a> `secondsUntilExpiration?` | <code>number \| null</code> | The number of seconds until the API key expires. Defaults to `null` (never expires). |
| <a id="subject"></a> `subject` | `string` | The user or Organization ID to associate the API key with. |
