The `useOrganizationCreationDefaults()` hook retrieves the organization creation defaults for the current user.

## Parameters

| Parameter                  | Type                                                                                     | Description                                                                                         |
| -------------------------- | ---------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------- |
| `params`                   | [`UseOrganizationCreationDefaultsParams`](use-organization-creation-defaults-params.mdx) | -                                                                                                   |
| `params.enabled?`          | `boolean`                                                                                | If `true`, a request will be triggered when the hook is mounted. Defaults to `true`.                |
| `params.keepPreviousData?` | `boolean`                                                                                | If true, the previous data will be kept in the cache until new data is fetched. Defaults to `true`. |

## Returns

[`UseOrganizationCreationDefaultsReturn`](use-organization-creation-defaults-return.mdx)

## Example

### Basic usage

```tsx
import { useOrganizationCreationDefaults } from "@clerk/clerk-react";

export default function CreateOrganizationForm() {
  const { data, isLoading } = useOrganizationCreationDefaults();

  if (isLoading) return <div>Loading...</div>;

  return (
    <form>
      <input defaultValue={data?.form.name} placeholder="Organization name" />
      <input defaultValue={data?.form.slug} placeholder="Slug" />
      <button type="submit">Create</button>
    </form>
  );
}
```
