### `createOrganization()`

Creates an Organization programmatically, adding the current user as admin. Returns an [`Organization`](/docs/reference/objects/organization) object.

> [!NOTE]
> For React-based apps, consider using the [`<CreateOrganization />`](/docs/reference/components/organization/create-organization) component.

```typescript
function createOrganization(params: CreateOrganizationParams): Promise<OrganizationResource>
```

#### `CreateOrganizationParams`


| Property | Type | Description |
| ------ | ------ | ------ |
| <a id="name"></a> `name` | `string` | The name of the Organization. |
| <a id="slug"></a> `slug?` | `string` | The slug of the Organization. |
