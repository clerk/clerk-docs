## cancelSubscriptionItem()

Cancels the given Subscription Item.

### Parameters

| Parameter            | Type                                 | Description                                                                                                                                                |
| -------------------- | ------------------------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `subscriptionItemId` | `string`                             | The ID of the Subscription Item to cancel.                                                                                                                 |
| `params?`            | <code>\{ endNow?: boolean; \}</code> | The parameters for the request.                                                                                                                            |
| `params?.endNow?`    | `boolean`                            | Whether the Subscription Item should be canceled immediately. If `false`, the Subscription Item will be canceled at the end of the current billing period. |

### Returns

`Promise`\<[`BillingSubscriptionItem`](../billing-subscription-item.mdx)\> — The cancelled [`BillingSubscriptionItem`](/docs/reference/backend/types/billing-subscription-item) object.

## extendSubscriptionItemFreeTrial()

Extends the free trial for the given Subscription Item.

### Parameters

| Parameter            | Type                               | Description                                                                                                             |
| -------------------- | ---------------------------------- | ----------------------------------------------------------------------------------------------------------------------- |
| `subscriptionItemId` | `string`                           | The ID of the Subscription Item to extend the free trial for.                                                           |
| `params`             | <code>\{ extendTo: Date; \}</code> | The parameters for the request.                                                                                         |
| `params.extendTo`    | `Date`                             | The date to extend the free trial to. Must be in the future and not more than 365 days from the current trial end date. |

### Returns

`Promise`\<[`BillingSubscriptionItem`](../billing-subscription-item.mdx)\> — The updated [`BillingSubscriptionItem`](/docs/reference/backend/types/billing-subscription-item) object.

## getPlanList()

Gets the list of Billing Plans for the instance. By default, the list is returned in descending order by creation date (newest first).

### Parameters

| Parameter            | Type                                               | Description                                                                                                                                                                            |
| -------------------- | -------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `params?`            | [`GetPlanListParams`](../get-plan-list-params.mdx) | -                                                                                                                                                                                      |
| `params?.limit?`     | `number`                                           | Maximum number of items returned per request. Must be an integer greater than zero and less than `501`. Can be used for paginating the results together with offset. Defaults to `10`. |
| `params?.offset?`    | `number`                                           | Skip the first `offset` items when paginating. Needs to be an integer greater or equal to zero. To be used in conjunction with `limit`. Defaults to `0`.                               |
| `params?.payerType?` | <code>"org" \| "user"</code>                       | Filters plans by the type of payer.                                                                                                                                                    |

### Returns

`Promise`\<[`PaginatedResourceResponse`](../paginated-resource-response.mdx)\<<code>[BillingPlan](../billing-plan.mdx)[]</code>\>\> — A [`PaginatedResourceResponse`](/docs/reference/backend/types/paginated-resource-response) object with a `data` property containing an array of [`BillingPlan`](/docs/reference/backend/types/billing-plan) objects and a `totalCount` property containing the total number of Billing Plans for the instance.
