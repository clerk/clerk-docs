---
title: '`revokeToken()`'
description: Use the revokeToken() method to revoke an OAuth application token.
---

Revokes both the OAuth access token and refresh token for the associated grant for the given [`OAuthApplication`](/docs/reference/backend/types/backend-oauth-application). The request may specify either token.

> [!IMPORTANT]
> JWT access tokens cannot be revoked. Only opaque OAuth access tokens and refresh tokens can be revoked.

```ts
function revokeToken(params: RevokeOAuthApplicationTokenParams): Promise<void>
```

## `RevokeOAuthApplicationTokenParams`

<Properties>
  - `oauthApplicationId`
  - `string`

  The ID of the OAuth application for which to revoke the token.

  ---

  - `token`
  - `string`

  The opaque OAuth access token or refresh token to revoke. All tokens associated with the same OAuth grant are also revoked.
</Properties>

## Example

<Include src="_partials/backend/usage" />

```tsx
const oauthApplicationId = 'oauthapp_123'
const token = 'oat_123'

await clerkClient.oauthApplications.revokeToken({ oauthApplicationId, token })
```

## Backend API (BAPI) endpoint

This method in the SDK is a wrapper around the BAPI endpoint `POST/oauth_applications/{oauth_application_id}/revoke_token`. See the [BAPI reference](/docs/reference/backend-api/tag/oauth-applications/POST/oauth_applications/%7Boauth_application_id%7D/revoke_token){{ target: '_blank' }} for more information.
