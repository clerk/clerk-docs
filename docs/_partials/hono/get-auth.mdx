The following example uses `getAuth()` to protect a route and load the user's data. If the user is authenticated, their `userId` is passed to [`clerk.users.getUser()`](/docs/reference/backend/user/get-user){{ target: '_blank' }} to get the current user's [`User`](/docs/reference/objects/user){{ target: '_blank' }} object. If not authenticated, the request is rejected with a `401` status code.

```ts
import { clerkMiddleware, getAuth } from '@clerk/hono'
import { Hono } from 'hono'

const app = new Hono()

app.use('*', clerkMiddleware())

app.get('/protected', async (c) => {
  const { isAuthenticated, userId } = getAuth(c)

  if (!isAuthenticated) {
    return c.json({ error: 'User not authenticated' }, 401)
  }

  const clerk = c.get('clerk')
  const user = await clerk.users.getUser(userId)

  return c.json({ user })
})

export default app
```
