### `isPrimaryIdentification()`

A check whether or not the given resource is the primary identifier for the user.

```typescript
function isPrimaryIdentification(ident: Web3WalletResource | EmailAddressResource | PhoneNumberResource): boolean
```

#### Parameters


| Parameter | Type | Description |
| ------ | ------ | ------ |
| `ident` | <code>[Web3WalletResource](/docs/reference/types/web3-wallet) \| [EmailAddressResource](/docs/reference/types/email-address) \| [PhoneNumberResource](/docs/reference/types/phone-number)</code> | The resource checked against the user to see if it is the primary identifier. |
