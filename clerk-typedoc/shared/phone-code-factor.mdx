## Properties

| Property                                     | Type                             | Description                                                                                                          |
| -------------------------------------------- | -------------------------------- | -------------------------------------------------------------------------------------------------------------------- |
| <a id="channel"></a> `channel?`              | <code>"sms" \| "whatsapp"</code> | The channel used for the phone code factor.                                                                          |
| <a id="default"></a> `default?`              | `boolean`                        | Indicates whether the phone number is set as the default identifier.                                                 |
| <a id="phonenumberid"></a> `phoneNumberId`   | `string`                         | The ID of the phone number used for the phone code factor.                                                           |
| <a id="primary"></a> `primary?`              | `boolean`                        | Indicates whether the phone number is set as the primary phone number, as multiple can be added to a user's profile. |
| <a id="safeidentifier"></a> `safeIdentifier` | `string`                         | The identifier provided by the user, but masked for security reasons.                                                |
| <a id="strategy"></a> `strategy`             | `"phone_code"`                   | The strategy type.                                                                                                   |
