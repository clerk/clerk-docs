# Add custom items and links to the `<UserButton />` component

The [`<UserButton />`](/docs/components/user/user-button) component supports the addition of custom items and use of external links in user button menu.

For the sake of this guide, examples are written for Next.js App Router, but they are supported by any React meta framework, such as Remix or Gatsby.

## Add a custom menu item to `<UserButton />`

Custom menu items can be rendered inside the `<UserButton />` component and provide a way to incorporate app-specific settings or additional functionality.

To add a custom menu item to the `<UserButton />` component, use the `<UserButton.MenuItems />` component, and within it, include the `<UserButton.Action>` or `<UserButton.Link>` component, depending on your use case.

### Props

`<UserButton.Action />` accept the following props:

| Name | Type | Description |
| - | - | - |
| `label` | `string` | The name that will be displayed in the menu of the user button. |
| `labelIcon` | `React.ReactElement` | An icon displayed next to the label in the menu. |
| `open?` | `string` | The path segment that will be used to open user profile modal to a specific page. |
| `onClick?` | `void` | A function to be called when the menu item is clicked. |

### Example

The following example demonstrates that you can render actions in the `<UserButton.MenuItems />` component.

<Tabs items={["<UserButton />"]}>
  <Tab>
    ```tsx filename="/app/page.tsx"
    'use client'

    import { UserButton } from '@clerk/nextjs'

    const DotIcon = () => {
      return (
        <svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 512 512" fill="currentColor">
          <path d="M256 512A256 256 0 1 0 256 0a256 256 0 1 0 0 512z" />
        </svg>
      )
    }

    export default function Home() {
      return (
        <header>
          <UserButton>
            <UserButton.MenuItems>
              <UserButton.Action
                label="Open chat"
                labelIcon={<DotIcon />}
                onClick={() => alert('init chat')}
              />
            </UserButton.MenuItems>
          </UserButton>
        </header>
      )
    }
    ```
  </Tab>
</Tabs>

### Example

The following example demonstrates that you can render actions in the `<UserButton.MenuItems />` component and open a user profile modal page.

    'use client'

    import { UserButton } from '@clerk/nextjs'

    const DotIcon = () => {

        <svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 512 512" fill="currentColor">

        </svg>
      )
    }

    export default function Home() {
      return (

## Add a custom link to <code>\<UserButtonÂ /></code>

You can add external links to the `<UserButton />` menu using the `<UserButton.Link />` component.

### Props

`<UserButton.Link />` accept the following props, all of which are _required_:

| Name | Type | Description |
| - | - | - |
| `label` | `string` | The name that will be displayed in the menu of the user button. |
| `labelIcon` | `React.ReactElement` | An icon displayed next to the label in the menu. |
| `href` | `string` | The path segment that will be used to navigate to the custom page. |

### Example

The following example adds a link to the menu of the `<UserButton />` component.

<Tabs items={["<UserButton />"]}>
  <Tab>
    ```tsx filename="/app/page.tsx"
    'use client'

    import { UserButton } from '@clerk/nextjs'

    const DotIcon = () => {
      return (
        <svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 512 512" fill="currentColor">
          <path d="M256 512A256 256 0 1 0 256 0a256 256 0 1 0 0 512z" />
        </svg>
      )
    }

    export default function Home() {
      return (
        <header>
          <UserButton>
            <UserButton.MenuItems>
              <UserButton.Link
                label="Create organization"
                labelIcon={<DotIcon />}
                href="/create-organization"
              />
            </UserButton.MenuItems>
          </UserButton>
        </header>
      )
    }
    ```
  </Tab>
</Tabs>

## Reorder default items

If you want to reorder the default menu items, `Manage account` and `Sign out`, set the `label` prop to `'manageAccount'` or `'signOut'`. This will target the existing default item and allow you to rearrange it.

Note STH

<Tabs items={["<UserButton />"]}>
  <Tab>
    ```tsx filename="/app/page.tsx"
    'use client'

    import { UserButton } from '@clerk/nextjs'

    const DotIcon = () => {
      return (
        <svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 512 512" fill="currentColor">
          <path d="M256 512A256 256 0 1 0 256 0a256 256 0 1 0 0 512z" />
        </svg>
      )
    }

    export default function Home() {
      return (
        <header>
          <UserButton>
            <UserButton>
              <UserButton.MenuItems>
                <UserButton.Action label="signOut" />
                <UserButton.Link
                  label="Create organization"
                  labelIcon={<DotIcon />}
                  href="/create-organization"
                />
                <UserButton.Action label="manageAccount" />
              </UserButton.MenuItems>
            </UserButton>
          </UserButton>
        </header>
      )
    }
    ```
  </Tab>
</Tabs>

With the above example, the `<UserButton />` menu items will be in the following order:

1. Sign out
1. Create organization
1. Manage account
