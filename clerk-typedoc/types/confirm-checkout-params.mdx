**`Experimental`**

The `confirm()` method accepts the following parameters. **Only one of `paymentSourceId`, `paymentToken`, or `useTestCard` should be provided.**

This is an experimental API for the Billing feature that is available under a public beta, and the API is subject to change. It is advised to pin the SDK version and the clerk-js version to avoid breaking changes.

## Type declaration

| Name               | Type     | Description                                                |
| ------------------ | -------- | ---------------------------------------------------------- |
| `paymentSourceId?` | `string` | The ID of a saved payment source to use for this checkout. |

| Name            | Type                                    | Description                                                                                                                     |
| --------------- | --------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------- |
| `gateway?`      | [`PaymentGateway`](payment-gateway.mdx) | The payment gateway to use. For example, `'stripe'` or `'paypal'`. **Required** if `paymentToken` or `useTestCard` is provided. |
| `paymentToken?` | `string`                                | A token representing payment details, usually from a payment form. **Requires** `gateway` to be provided.                       |

| Name           | Type                                    | Description                                                                                                                     |
| -------------- | --------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------- |
| `gateway?`     | [`PaymentGateway`](payment-gateway.mdx) | The payment gateway to use. For example, `'stripe'` or `'paypal'`. **Required** if `paymentToken` or `useTestCard` is provided. |
| `useTestCard?` | `boolean`                               | If true, uses a test card for the checkout. **Requires** `gateway` to be provided.                                              |
