## create()

Creates an Agent Task that generates a URL which, when visited, creates a session for the specified user. This is useful for automated testing or agent-driven flows where full authentication isn't practical.

### Parameters

| Parameter | Type                                                       |
| --------- | ---------------------------------------------------------- |
| `params`  | [`CreateAgentTaskParams`](../create-agent-task-params.mdx) |

### Returns

`Promise`\<[`AgentTask`](../agent-task.mdx)\> — The created [`AgentTask`](/docs/reference/backend/types/backend-agent-task) object.

---

## revoke()

Revokes the given Agent Task.

### Parameters

| Parameter     | Type     | Description                         |
| ------------- | -------- | ----------------------------------- |
| `agentTaskId` | `string` | The ID of the Agent Task to revoke. |

### Returns

`Promise`\<`Omit`\<[`AgentTask`](../agent-task.mdx), `"url"`\>\> — The revoked [`AgentTask`](/docs/reference/backend/types/backend-agent-task) object.
