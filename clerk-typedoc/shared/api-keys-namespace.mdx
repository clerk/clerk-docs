## create()

Creates a new API key.

### Parameters

| Parameter | Type                                              |
| --------- | ------------------------------------------------- |
| `params`  | [`CreateAPIKeyParams`](create-api-key-params.mdx) |

### Returns

`Promise`\<[`APIKeyResource`](api-key-resource/api-key-resource.mdx)\> — An [`APIKeyResource`](/docs/reference/types/api-key-resource) object that includes the `secret` property.

> [!WARNING]
> Make sure to store the API key secret immediately after creation, as it will not be available again.

---

## getAll()

Gets a paginated list of API keys for the current user or organization.

### Parameters

| Parameter              | Type                                          | Description                                                                                                                                                         |
| ---------------------- | --------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `params?`              | [`GetAPIKeysParams`](get-api-keys-params.mdx) | -                                                                                                                                                                   |
| `params?.initialPage?` | `number`                                      | A number that specifies which page to fetch. For example, if `initialPage` is set to `10`, it will skip the first 9 pages and fetch the 10th page. Defaults to `1`. |
| `params?.pageSize?`    | `number`                                      | A number that specifies the maximum number of results to return per page. Defaults to `10`.                                                                         |

### Returns

`Promise`\<[`ClerkPaginatedResponse`](/docs/reference/types/clerk-paginated-response)\<[`APIKeyResource`](api-key-resource/api-key-resource.mdx)\>\> — A [`ClerkPaginatedResponse`](/docs/reference/types/clerk-paginated-response) of [`APIKeyResource`](/docs/reference/types/api-key-resource) objects.
