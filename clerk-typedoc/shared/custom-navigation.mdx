A function used to navigate to a given URL after certain steps in the Clerk processes.

## Parameters

| Parameter            | Type                                                             | Description                                                                |
| -------------------- | ---------------------------------------------------------------- | -------------------------------------------------------------------------- |
| `to`                 | `string`                                                         | The URL or relative path to navigate to.                                   |
| `options?`           | <code>\{ metadata?: RouterMetadata; replace?: boolean; \}</code> | Optional configuration.                                                    |
| `options?.metadata?` | `RouterMetadata`                                                 | Router metadata.                                                           |
| `options?.replace?`  | `boolean`                                                        | If `true`, replace the current history entry instead of pushing a new one. |

## Returns

<code>Promise\<unknown\> \| void</code>
