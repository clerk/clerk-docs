An interface that represents a task associated with a session.

```tsx
export interface SessionTask {
  key: 'org'
}
```

<Properties>
  - `key`
  - `'orgs'`

  A enum that includes a unique identifier for the task.

  - `'orgs'`: Related with the [Disable personal workspaces](/docs/organizations/overview#disable-personal-workspaces) setting.
</Properties>
