Retrieves a user's Billing Subscription. Returns a [`BillingSubscription`](/docs/reference/backend/types/billing-subscription).

```ts
function getUserBillingSubscription(userId: string): Promise<BillingSubscription>
```

## Example

<Include src="_partials/backend/usage" />

```tsx
const userId = 'user_123'

const subscription = await clerkClient.billing.getUserBillingSubscription(userId)
```

## Backend API (BAPI) endpoint

This method in the SDK is a wrapper around the BAPI endpoint `GET /users/{user_id}/billing/subscription`. See the [BAPI reference](/docs/reference/backend-api/tag/billing/GET/users/%7Buser_id%7D/billing/subscription){{ target: '_blank' }} for more information.
