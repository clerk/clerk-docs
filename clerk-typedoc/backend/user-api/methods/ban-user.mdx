Marks the given [`User`](/docs/reference/backend/types/backend-user) as banned, which means that all their sessions are revoked and they are not allowed to sign in again.

```typescript
function banUser(userId: string): Promise<User>
```

## Parameters


| Parameter | Type | Description |
| ------ | ------ | ------ |
| `userId` | `string` | The ID of the user to ban. |
