The `OrganizationDomain` object is the model around an Organization's [Verified Domain](/docs/guides/organizations/add-members/verified-domains).

## Properties

| Property                                                                     | Type                                                                                                                                                | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| ---------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| <a id="affiliationemailaddress"></a> `affiliationEmailAddress`               | <code>null \| string</code>                                                                                                                         | The email address used to verify the affiliation with the domain, or `null` if none has been provided.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| <a id="attemptaffiliationverification"></a> `attemptAffiliationVerification` | <code>(params: [AttemptAffiliationVerificationParams](attempt-affiliation-verification-params.mdx)) => Promise\<OrganizationDomainResource\></code> | Completes the affiliation verification flow by validating the code sent to the affiliation email address.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| <a id="createdat"></a> `createdAt`                                           | `Date`                                                                                                                                              | The date and time when the domain was created.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| <a id="delete"></a> `delete`                                                 | <code>() => Promise\<void\></code>                                                                                                                  | Deletes the Verified Domain.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| <a id="enrollmentmode"></a> `enrollmentMode`                                 | <code>"manual_invitation" \| "automatic_invitation" \| "automatic_suggestion"</code>                                                                | The enrollment mode that determines how matching users are added to the Organization. <ul> <li>`manual_invitation`: No automatic enrollment. Users with a matching email domain are not given any [invitation](/docs/guides/organizations/add-members/verified-domains#automatic-invitations) or [suggestion](/docs/guides/organizations/add-members/verified-domains#automatic-suggestions); an admin must invite them manually.</li> <li>`automatic_invitation`: Users with a matching email domain automatically receive a pending [invitation](/docs/reference/types/organization-invitation) (assigned the Organization's default role) which they can accept to join.</li> <li>`automatic_suggestion`: Users with a matching email domain automatically receive a [suggestion](/docs/guides/organizations/add-members/verified-domains#automatic-suggestions) to join, which they can request.</li> </ul> |
| <a id="id"></a> `id`                                                         | `string`                                                                                                                                            | The unique identifier for the Verified Domain.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| <a id="name"></a> `name`                                                     | `string`                                                                                                                                            | The domain name, for example `clerk.com`.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| <a id="organizationid"></a> `organizationId`                                 | `string`                                                                                                                                            | The ID of the Organization that the Verified Domain belongs to.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| <a id="prepareaffiliationverification"></a> `prepareAffiliationVerification` | <code>(params: [PrepareAffiliationVerificationParams](prepare-affiliation-verification-params.mdx)) => Promise\<OrganizationDomainResource\></code> | Begins the affiliation verification flow by sending a verification code to the provided email address.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| <a id="totalpendinginvitations"></a> `totalPendingInvitations`               | `number`                                                                                                                                            | The total number of pending [invitations](/docs/reference/types/organization-invitation) associated with this domain.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| <a id="totalpendingsuggestions"></a> `totalPendingSuggestions`               | `number`                                                                                                                                            | The total number of pending [suggestions](/docs/reference/types/organization-suggestion) associated with this domain.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| <a id="updatedat"></a> `updatedAt`                                           | `Date`                                                                                                                                              | The date and time when the domain was last updated.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| <a id="updateenrollmentmode"></a> `updateEnrollmentMode`                     | <code>(params: [UpdateEnrollmentModeParams](update-enrollment-mode-params.mdx)) => Promise\<OrganizationDomainResource\></code>                     | Updates the enrollment mode of the Verified Domain.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| <a id="verification"></a> `verification`                                     | <code>null \| [OrganizationDomainVerification](organization-domain-verification.mdx)</code>                                                         | The verification details for the domain, or `null` if the domain has not been verified.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
