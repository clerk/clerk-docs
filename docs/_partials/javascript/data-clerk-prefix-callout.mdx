> [!IMPORTANT]
> Any Clerk option passed as a script tag attribute must be prefixed with `data-clerk-`. For example, `publishableKey` becomes `data-clerk-publishable-key`, `proxyUrl` becomes `data-clerk-proxy-url`, and `domain` becomes `data-clerk-domain`. See the [`Clerk`](/docs/reference/objects/clerk) object for the full list of available options.
