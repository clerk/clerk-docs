Updates the [restriction](/docs/guides/secure/restricting-access) settings for the current instance.

Returns the updated [`InstanceRestrictions`](/docs/reference/backend/types/backend-instance-restrictions) object.

```typescript
function updateRestrictions(params: UpdateRestrictionsParams): Promise<InstanceRestrictions>
```

## `UpdateRestrictionsParams`


| Property | Type | Description |
| ------ | ------ | ------ |
| <a id="allowlist"></a> `allowlist?` | <code>boolean \| null</code> | Whether the instance should have [**Allowlist**](/docs/guides/secure/restricting-access#allowlist) enabled. |
| <a id="blockdisposableemaildomains"></a> `blockDisposableEmailDomains?` | <code>boolean \| null</code> | Whether the instance should have [**Block sign-ups that use disposable email domains**](/docs/guides/secure/restricting-access#block-sign-ups-that-use-disposable-email-addresses) enabled. |
| <a id="blockemailsubaddresses"></a> `blockEmailSubaddresses?` | <code>boolean \| null</code> | Whether the instance should have [**Block email subaddresses**](/docs/guides/secure/restricting-access#block-email-subaddresses) enabled. |
| <a id="blocklist"></a> `blocklist?` | <code>boolean \| null</code> | Whether the instance should have [**Blocklist**](/docs/guides/secure/restricting-access#blocklist) enabled. |
| <a id="ignoredotsforgmailaddresses"></a> `ignoreDotsForGmailAddresses?` | <code>boolean \| null</code> | Whether the instance should [ignore dots for Gmail addresses](/docs/guides/secure/restricting-access#block-email-subaddresses). |
