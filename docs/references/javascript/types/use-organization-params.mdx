# `useOrganizationParams`

Parameters for the [`useOrganization()`](/docs/references/react/use-organization) hook.

```typescript
type useOrganizationParams = {
  invitationList: {
    // Pagination params
    limit?: number;
    offset?: number;
    },
  membershipRequests: true
    | (commonPaginationParams & {
        status?: OrganizationInvitationStatus; // e.g. "pending"
      });
  memberships: true
    | (commonPaginationParams & {
        role?: MembershipRole[]; // e.g. ["basic_member", "admin"]
      });
  domains: true
    | (commonPaginationParams & {
        enrollmentMode?: OrganizationEnrollmentMode; // e.g. "automatic_invitation"
      });
};

type commonPaginationParams = {
  initialPage?: number;
  pageSize?: number;
  infinite?: boolean;
  keepPreviousData?: boolean;
}
```
