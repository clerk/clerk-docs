## Usage

<Include src="_partials/backend/usage" />

```tsx
const response = await clerkClient.redirectUrls.createRedirectUrl({
  url: 'https://example.com',
})
```

## Backend API (BAPI) endpoint

This method in the SDK is a wrapper around the BAPI endpoint `POST/redirect_urls`. See the [BAPI reference](/docs/reference/backend-api/tag/redirect-urls/POST/redirect_urls){{ target: '_blank' }} for more information.
