The `BillingPerUnitTotalTier` type represents the cost breakdown for a single tier in checkout totals.

## Properties

| Property                               | Type                                                               | Description                                                   |
| -------------------------------------- | ------------------------------------------------------------------ | ------------------------------------------------------------- |
| <a id="feeperblock"></a> `feePerBlock` | [`BillingMoneyAmount`](/docs/reference/types/billing-money-amount) | The fee charged per block for this tier.                      |
| <a id="quantity"></a> `quantity`       | <code>null \| number</code>                                        | The quantity billed within this tier. `null` means unlimited. |
| <a id="total"></a> `total`             | [`BillingMoneyAmount`](/docs/reference/types/billing-money-amount) | The total billed amount for this tier.                        |
