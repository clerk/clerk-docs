---
title: '`Passkey`'
description: Create and manage passkeys for a user.
sdk: ios
---

`Passkey` represents a passkey registered for the signed-in user.

## Create a passkey

```swift
@Environment(Clerk.self) private var clerk

if let user = clerk.user {
  let passkey = try await user.createPasskey()
  print(passkey.id)
}
```

## Rename a passkey

```swift
let updated = try await passkey.update(name: "My Mac")
```

## Verify a passkey

```swift
_ = try await passkey.attemptVerification(credential: credential)
```

## Delete a passkey

```swift
_ = try await passkey.delete()
```

## Key properties

<Properties>
  - `name`
  - `String`

  Passkey name.

  ---

  - `verification`
  - `Verification?`

  Verification details.

  ---

  - `createdAt`
  - `Date`

  Creation timestamp.

  ---

  - `lastUsedAt`
  - `Date?`

  Last time the passkey was used.
</Properties>
