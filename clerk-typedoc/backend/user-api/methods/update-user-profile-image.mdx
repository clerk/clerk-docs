Updates the profile image for the given user. To remove the profile image, see [`deleteUserProfileImage()`](/docs/reference/backend/user/delete-user-profile-image).

Returns the updated [`User`](/docs/reference/backend/types/backend-user).

```typescript
function updateUserProfileImage(userId: string, params: { file: Blob | File }): Promise<User>
```

## Parameters


| Parameter | Type | Description |
| ------ | ------ | ------ |
| `userId` | `string` | The ID of the user to update the profile image for. |
| `params` | <code>\{ file: Blob \| File; \}</code> | The file to set as the user's profile image. |
