Deletes all of a user's TOTPs.

```ts
function deleteUserTOTP(userId: string): Promise<UserID>
```

## Example

<Include src="_partials/backend/usage" />

```tsx
const userId = 'user_123'

const response = await clerkClient.users.deleteUserTOTP(userId)
```

## Backend API (BAPI) endpoint

This method in the SDK is a wrapper around the BAPI endpoint `DELETE/users/{user_id}/totp`. See the [BAPI reference](/docs/reference/backend-api/tag/users/delete/users/%7Buser_id%7D/totp){{ target: '_blank' }} for more information.
