## list()

Gets a list of OAuth applications for the instance. By default, the list is returned in descending order by creation date (newest first).

### Parameters

| Parameter         | Type                                                                         | Description                                                                                                                                                                            |
| ----------------- | ---------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `params`          | [`GetOAuthApplicationListParams`](../get-o-auth-application-list-params.mdx) | The parameters to get the OAuth applications with.                                                                                                                                     |
| `params.limit?`   | `number`                                                                     | Maximum number of items returned per request. Must be an integer greater than zero and less than `501`. Can be used for paginating the results together with offset. Defaults to `10`. |
| `params.offset?`  | `number`                                                                     | Skip the first `offset` items when paginating. Needs to be an integer greater or equal to zero. To be used in conjunction with `limit`. Defaults to `0`.                               |
| `params.orderBy?` | `WithSign`\<<code>"name" \| "created_at"</code>\>                            | Returns OAuth applications in a particular order. Prefix a value with `+` to sort in ascending order, or `-` to sort in descending order. Defaults to `-created_at`.                   |

### Returns

`Promise`\<[`PaginatedResourceResponse`](../paginated-resource-response.mdx)\<<code>[OAuthApplication](../o-auth-application.mdx)[]</code>\>\> — A [`PaginatedResourceResponse`](/docs/reference/backend/types/paginated-resource-response) object with a `data` property containing an array of [`OAuthApplication`](/docs/reference/backend/types/backend-oauth-application) objects and a `totalCount` property containing the total number of OAuth applications.

## rotateSecret()

Rotates the secret of the given OAuth application. When the client secret is rotated, ensure that you update it in your authorized OAuth clients.

### Parameters

| Parameter            | Type     | Description                                              |
| -------------------- | -------- | -------------------------------------------------------- |
| `oauthApplicationId` | `string` | The ID of the OAuth application to rotate the secret of. |

### Returns
