The `FeatureResource` type represents a feature of a plan.

## Properties

| Property                               | Type                        | Description                                                                  |
| -------------------------------------- | --------------------------- | ---------------------------------------------------------------------------- |
| <a id="avatarurl"></a> `avatarUrl`     | <code>null \| string</code> | The URL of the feature's avatar image, or `null` if not set.                 |
| <a id="description"></a> `description` | <code>null \| string</code> | A short description of what the feature provides, or `null` if not provided. |
| <a id="id"></a> `id`                   | `string`                    | The unique identifier for the feature.                                       |
| <a id="name"></a> `name`               | `string`                    | The display name of the feature.                                             |
| <a id="pathroot"></a> `pathRoot`       | `string`                    | The root path of the resource.                                               |
| <a id="slug"></a> `slug`               | `string`                    | A unique, URL-friendly identifier for the feature.                           |
