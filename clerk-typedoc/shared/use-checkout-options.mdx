| Property                                    | Type                                  | Description                                                                                             |
| ------------------------------------------- | ------------------------------------- | ------------------------------------------------------------------------------------------------------- |
| <a id="for"></a> `for?`                     | <code>"user" \| "organization"</code> | Specifies if the checkout is for an Organization. Defaults to `'user'`.                                 |
| <a id="planid"></a> `planId`                | `string`                              | The ID of the Subscription Plan to check out (e.g. `cplan_xxx`).                                        |
| <a id="planperiod"></a> `planPeriod`        | <code>"month" \| "annual"</code>      | The billing period for the Plan.                                                                        |
| <a id="priceid"></a> `priceId?`             | `string`                              | The specific price ID to check out for, used when the desired price ID is not the current default price |
| <a id="seatsquantity"></a> `seatsQuantity?` | `number`                              | The number of total seats to check out for                                                              |
