Updates the given OAuth application.

Returns the updated [`OAuthApplication`](/docs/reference/backend/types/backend-oauth-application) object.

```typescript
function update(params: UpdateOAuthApplicationParams): Promise<OAuthApplication>
```

## `UpdateOAuthApplicationParams`


| Property | Type | Description |
| ------ | ------ | ------ |
| <a id="name"></a> `name` | `string` | A descriptive name for the OAuth application (e.g., "My Web App", "My Mobile App"). Maximum length: 256 characters. |
| `oauthApplicationId` | `string` | The ID of the OAuth application to update. |
| <a id="public"></a> `public?` | <code>boolean \| null</code> | Whether the OAuth application is public. If `true`, the Proof Key of Code Exchange (PKCE) flow can be used. |
| <a id="redirecturis"></a> `redirectUris?` | <code>string[] \| null</code> | An array of redirect URIs for the OAuth application. |
| <a id="scopes"></a> `scopes?` | <code>string \| null</code> | Scopes for the OAuth application that dictate the user payload of the OAuth user info endpoint. Available scopes are `profile`, `email`, `public_metadata`, `private_metadata`. Provide the requested scopes as a string, separated by spaces. |
