<Steps>
  ### Install all skills with a single command

  ```bash
  npx skills add clerk/skills
  ```

  Skills work with most agents including Claude Code, Cursor, Windsurf, GitHub Copilot, Codex, and Gemini CLI.

  [Learn more about Clerk Skills](/docs/guides/ai/skills)

  ### Ask AI agent to add Clerk auth

  | You say | Skill used |
  | - | - |
  | "Add Clerk auth to my Next.js app" | `/clerk-setup` |

  ### Sign up as your first user

  Once AI has finished, you need to sign up as your first user:

    1. Run `npm run dev` in your terminal to run your project.
    1. View your app at [http://localhost:3000](http://localhost:3000).
    1. Click **Sign up** to experience your end-to-end authentication flow.

  ### Configure application and create your Clerk account

  This prompt quickstart auto-generates temporary API keys so you can test Clerk right away.

  Claim your keys with the callout in your app, which will allow you to create or log in to Clerk. Or manually [sign up](https://dashboard.clerk.com/sign-up) to link your keys, customize your auth settings, and view more features.
