Represents information about an image.

## Properties

| Property                           | Type                        | Description                         |
| ---------------------------------- | --------------------------- | ----------------------------------- |
| <a id="id"></a> `id?`              | `string`                    | The unique identifier of the image. |
| <a id="name"></a> `name`           | <code>null \| string</code> | The name of the image.              |
| <a id="publicurl"></a> `publicUrl` | <code>null \| string</code> | The public URL of the image.        |
