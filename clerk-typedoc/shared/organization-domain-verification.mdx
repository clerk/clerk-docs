The `OrganizationDomainVerification` object holds the affiliation verification details of an Organization's [Verified Domain](/docs/guides/organizations/add-members/verified-domains). Affiliation proves that the current user controls an email address that belongs to the domain.

This is the object referenced by both `affiliationVerification` and the deprecated `verification` property of the `OrganizationDomainResource` object.

## Properties

| Property                           | Type                                    | Description                                                                                       |
| ---------------------------------- | --------------------------------------- | ------------------------------------------------------------------------------------------------- |
| <a id="attempts"></a> `attempts`   | `number`                                | The number of verification attempts that have been made for the current affiliation verification. |
| <a id="expiresat"></a> `expiresAt` | `Date`                                  | The date and time when the current affiliation verification attempt expires.                      |
| <a id="status"></a> `status`       | <code>"unverified" \| "verified"</code> | The current status of the affiliation verification.                                               |
| <a id="strategy"></a> `strategy`   | `"email_code"`                          | The strategy used to verify affiliation with the domain.                                          |
