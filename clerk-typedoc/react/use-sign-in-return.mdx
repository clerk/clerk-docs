## Returns

| Name          | Type                                                             | Description                                                                                                         |
| ------------- | ---------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------- |
| `errors`      | [`SignInErrors`](/docs/reference/types/errors)                   | The errors that occurred during the last fetch of the underlying `SignInFuture` resource.                           |
| `fetchStatus` | <code>"idle" \| "fetching"</code>                                | The fetch status of the underlying `SignInFuture` resource.                                                         |
| `signIn`      | [`SignInFutureResource`](/docs/reference/objects/sign-in-future) | An instance representing the currently active `SignInFuture`, with new APIs designed specifically for custom flows. |
