## createSignInToken()

Creates a new sign-in token for the given user. By default, sign-in tokens expire in 30 days. You can optionally specify a custom expiration time in seconds using the `expiresInSeconds` parameter.

### Parameters

| Parameter | Type                                                              |
| --------- | ----------------------------------------------------------------- |
| `params`  | [`CreateSignInTokensParams`](../create-sign-in-tokens-params.mdx) |

### Returns

`Promise`\<[`SignInToken`](../sign-in-token.mdx)\> — The created [`SignInToken`](/docs/reference/backend/types/backend-sign-in-token) object.

---

## revokeSignInToken()

Revokes the given sign-in token.

### Parameters

| Parameter       | Type     | Description                            |
| --------------- | -------- | -------------------------------------- |
| `signInTokenId` | `string` | The ID of the sign-in token to revoke. |

### Returns

`Promise`\<[`SignInToken`](../sign-in-token.mdx)\> — The revoked [`SignInToken`](/docs/reference/backend/types/backend-sign-in-token) object.
