The `useSession()` composable provides access to the current user's [`Session`](/docs/reference/objects/session) object, as well as helpers for setting the active session.

## Returns

<Properties>
  - `isLoaded`
  - `Ref<boolean>`

  A boolean that indicates whether Clerk has finished loading and initializing. Returns `false` during initialization.

  ---

  - `isSignedIn`
  - `Ref<boolean>`

  A boolean that indicates whether a user is currently signed in.

  ---

  - `session`
  - <code>Ref\<[Session](/docs/reference/objects/session)></code>

  The current session for the user.
</Properties>
