### `unmountSignUp()`

Unmount a sign up flow component from the target element.
If there is no component mounted at the target node, results in a noop.

```typescript
function unmountSignUp(targetNode: HTMLDivElement): void
```

#### Parameters


| Parameter | Type | Description |
| ------ | ------ | ------ |
| `targetNode` | `HTMLDivElement` | Target node to unmount the SignUp component from. |
