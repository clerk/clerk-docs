The Backend `Role` object represents an Organization [Role](/docs/guides/organizations/control-access/roles-and-permissions) that can be assigned to Organization members.

## Properties

| Property                                           | Type                                        | Description                                                                                              |
| -------------------------------------------------- | ------------------------------------------- | -------------------------------------------------------------------------------------------------------- |
| <a id="createdat"></a> `createdAt`                 | `number`                                    | The Unix timestamp when the Role was first created.                                                      |
| <a id="description"></a> `description`             | <code>null \| string</code>                 | A description of the Role.                                                                               |
| <a id="id"></a> `id`                               | `string`                                    | The unique identifier for the Role.                                                                      |
| <a id="iscreatoreligible"></a> `isCreatorEligible` | `boolean`                                   | Whether this Role is eligible to be an Organization creator Role.                                        |
| <a id="key"></a> `key`                             | `string`                                    | The unique key of the Role, in the format `org:role`.                                                    |
| <a id="name"></a> `name`                           | `string`                                    | The name of the Role.                                                                                    |
| <a id="permissions"></a> `permissions`             | <code>[Permission](permission.mdx)[]</code> | The [Permissions](/docs/guides/organizations/control-access/roles-and-permissions) assigned to the Role. |
| <a id="updatedat"></a> `updatedAt`                 | `number`                                    | The Unix timestamp when the Role was last updated.                                                       |
