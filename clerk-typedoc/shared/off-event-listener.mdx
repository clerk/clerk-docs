Unregisters an event listener for a specific Clerk event.

## Parameters

| Parameter | Type                  | Description                         |
| --------- | --------------------- | ----------------------------------- |
| `event`   | `E`                   | The event name to unsubscribe from. |
| `handler` | `EventHandler`\<`E`\> | The callback function to remove.    |

## Returns

`void`
