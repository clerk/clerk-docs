## createScope()

Creates a new machine scope, allowing the specified machine to access another machine. Maximum of 150 scopes per machine.

### Parameters

| Parameter     | Type     | Description                                                          |
| ------------- | -------- | -------------------------------------------------------------------- |
| `machineId`   | `string` | The ID of the machine that will have access to the target machine.   |
| `toMachineId` | `string` | The ID of the machine that will be accessible by the source machine. |

### Returns

`Promise`\<[`MachineScope`](../machine-scope.mdx)\> — The created [`MachineScope`](/docs/reference/backend/types/backend-machine-scope) object.

## list()

Gets a list of machines for the instance. By default, the list is returned in descending order by creation date (newest first).

### Parameters

| Parameter              | Type                                                     | Description                                                                                                                                                                            |
| ---------------------- | -------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `queryParams`          | [`GetMachineListParams`](../get-machine-list-params.mdx) | -                                                                                                                                                                                      |
| `queryParams.limit?`   | `number`                                                 | Maximum number of items returned per request. Must be an integer greater than zero and less than `501`. Can be used for paginating the results together with offset. Defaults to `10`. |
| `queryParams.offset?`  | `number`                                                 | Skip the first `offset` items when paginating. Needs to be an integer greater or equal to zero. To be used in conjunction with `limit`. Defaults to `0`.                               |
| `queryParams.orderBy?` | `WithSign`\<<code>"name" \| "created_at"</code>\>        | Filters machines in a particular order. Prefix a value with `+` to sort in ascending order, or `-` to sort in descending order. Defaults to `-created_at`.                             |
| `queryParams.query?`   | `string`                                                 | Filters machines by ID or name.                                                                                                                                                        |

### Returns

`Promise`\<[`PaginatedResourceResponse`](../paginated-resource-response.mdx)\<<code>[Machine](../machine.mdx)[]</code>\>\> — A [`PaginatedResourceResponse`](/docs/reference/backend/types/paginated-resource-response) object with a `data` property containing an array of [`Machine`](/docs/reference/backend/types/backend-machine) objects and a `totalCount` property containing the total number of machines for the instance.
