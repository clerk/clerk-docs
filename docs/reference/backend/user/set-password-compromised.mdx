title: '`setPasswordCompromised()`'

---

<Typedoc src="backend/user-api/methods/set-password-compromised" />

## Example

<Include src="_partials/backend/usage" />

```tsx
const userId = 'user_123'

const response = await clerkClient.users.setPasswordCompromised(userId, {
  revokeAllSessions: true,
})
```

## Backend API (BAPI) endpoint

This method in the SDK is a wrapper around the BAPI endpoint `POST/users/{user_id}/password/set_compromised`. See the [BAPI reference](/docs/reference/backend-api/tag/users/POST/users/%7Buser_id%7D/password/set_compromised){{ target: '_blank' }} for more information.
