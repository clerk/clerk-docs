### `resetPasswordPhoneCode.verifyCode()`

Verifies a password reset code sent with the [`resetPasswordPhoneCode.sendCode()`](/docs/reference/objects/sign-in-future#reset-password-phone-code-send-code) method. Will cause `signIn.status` to become `'needs_new_password'`. This is when you will call the [`resetPasswordPhoneCode.submitPassword()`](/docs/reference/objects/sign-in-future#reset-password-phone-code-submit-password) method to complete the password reset flow.

```typescript
function verifyCode(params: SignInFutureResetPasswordPhoneCodeVerifyParams): Promise<{ error: null | ClerkError }>
```

#### `SignInFutureResetPasswordPhoneCodeVerifyParams`


| Property | Type | Description |
| ------ | ------ | ------ |
| <a id="code"></a> `code` | `string` | The one-time code that was sent to the user. |
