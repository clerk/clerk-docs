---
title: '`createClerkHandler()`'
description: The createClerkHandler() is used to configure the Clerk for your TanStack Start application.
---

The `createClerkHandler()` function configures Clerk to handle authentication state for TanStack routes, allowing you to easily access user session information within your app.

## Usage

### Basic usage

The following example demonstrates how to use `createClerkHandler()` to configure Clerk in your TanStack Start application.

```tsx {{ filename: 'app/ssr.tsx' }}
import { createStartHandler, defaultStreamHandler } from '@tanstack/start/server'
import { getRouterManifest } from '@tanstack/start/router-manifest'
import { createRouter } from './router'
import { createClerkHandler } from '@clerk/tanstack-start/server'

export default createClerkHandler(
  createStartHandler({
    createRouter,
    getRouterManifest,
  }),
)(defaultStreamHandler)
```

### Advanced usage

The following example demonstrates how to configure `createClerkHandler` with your options without depending on the default environment variables or changing them dynamically.

```tsx {{ filename: 'app/ssr.tsx' }}
import { createStartHandler, defaultStreamHandler } from '@tanstack/start/server'
import { getRouterManifest } from '@tanstack/start/router-manifest'
import { createRouter } from './router'
import { createClerkHandler } from '@clerk/tanstack-start/server'

export default createClerkHandler(
  createStartHandler({
    createRouter,
    getRouterManifest,
  }),
  {
    secretKey: process.env.CLERK_SECRET_KEY_APP_1,
    publishableKey: process.env.CLERK_PUBLISHABLE_KEY_APP_2,
    signInUrl: '/my-sign-in',
    signUpUrl: '/my-sign-up',
  },
)(defaultStreamHandler)
```
