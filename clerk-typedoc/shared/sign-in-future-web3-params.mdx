## Properties

| Property                              | Type                                                                                                                                                          | Description                                                                                           |
| ------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------- |
| <a id="provider"></a> `provider`      | <code>"metamask" \| "base" \| "coinbase_wallet" \| "okx_wallet" \| "solana"</code>                                                                            | The Web3 wallet provider to use for the sign-in.                                                      |
| <a id="strategy"></a> `strategy`      | <code>"web3_metamask_signature" \| "web3_base_signature" \| "web3_coinbase_wallet_signature" \| "web3_okx_wallet_signature" \| "web3_solana_signature"</code> | The verification strategy to validate the user's sign-in request.                                     |
| <a id="walletname"></a> `walletName?` | `string`                                                                                                                                                      | **Required** when `provider` is set to `'solana'`. The name of the wallet to use for Solana sign-ins. |
