## Parameters

| Parameter              | Type                                                                                           | Description                                                                                                                                                         |
| ---------------------- | ---------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `params?`              | [`GetMembersParams`](get-members-params.mdx)                                                   | -                                                                                                                                                                   |
| `params?.initialPage?` | `number`                                                                                       | A number that specifies which page to fetch. For example, if `initialPage` is set to `10`, it will skip the first 9 pages and fetch the 10th page. Defaults to `1`. |
| `params?.pageSize?`    | `number`                                                                                       | A number that specifies the maximum number of results to return per page. Defaults to `10`.                                                                         |
| `params?.query?`       | `string`                                                                                       | The query to filter the users by.                                                                                                                                   |
| `params?.role?`        | <code>[OrganizationCustomRoleKey](/docs/reference/types/organization-custom-role-key)[]</code> | The [Role](/docs/guides/organizations/control-access/roles-and-permissions) to filter the users by.                                                                 |

`Promise`\<[`ClerkPaginatedResponse`](/docs/reference/types/clerk-paginated-response)\<[`OrganizationMembershipResource`](/docs/reference/types/organization-membership)\>\>
