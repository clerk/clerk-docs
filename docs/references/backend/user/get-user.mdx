Retrieves a single [`User`](/docs/references/backend/types/backend-user) by their ID, if the ID is valid.

```ts
function getUser(userId: string): Promise<User>
```

## Parameters

<Properties>
  - `userId`
  - `string`

  The ID of the user to retrieve.
</Properties>

## Example

> [!NOTE]
> Learn how to [get the `userId` and other properties](/docs/references/backend/overview#get-the-user-id-and-other-properties).

```tsx
const userId = 'user_123'

const response = await clerkClient.users.getUser(userId)
```

## Backend API (BAPI) endpoint

This method in the SDK is a wrapper around the BAPI endpoint `GET/users/{user_id}`. See the [BAPI reference](/docs/reference/backend-api/tag/Users#operation/GetUser){{ target: '_blank' }} for more details.
