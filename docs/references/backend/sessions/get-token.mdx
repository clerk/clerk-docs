# `getToken()`

Retrieves a specific session's token.

```tsx
const sessionId = 'my-session-id';

const session = await clerkClient.sessions.getToken(sessionId);
```

## Required parameters

| Name | Type | Description |
| --- | --- | --- |
| `sessionId` | `string` | The ID of the session to retrieve a token for. |
