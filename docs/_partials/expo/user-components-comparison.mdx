| Feature | `<UserButton />` | `<UserProfileView />` |
| - | - | - |
| Display | Small avatar button | Full profile interface |
| Use case | Header/navigation | Dedicated profile screen |
| Profile access | Opens as modal on tap | Renders directly |

Use `<UserButton />` when you want a compact trigger that opens profile management. Use `<UserProfileView />` when you want to embed the full profile interface in your app.
