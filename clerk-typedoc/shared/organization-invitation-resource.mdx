The `OrganizationInvitation` object is the model around [an invitation to join an Organization](/docs/guides/organizations/add-members/invitations).

## Properties

| Property                                     | Type                                                                                                           | Description                                                                                                                                                                                                                                                                                                                             |
| -------------------------------------------- | -------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| <a id="createdat"></a> `createdAt`           | `Date`                                                                                                         | The date when the invitation was created.                                                                                                                                                                                                                                                                                               |
| <a id="emailaddress"></a> `emailAddress`     | `string`                                                                                                       | The email address the invitation was sent to.                                                                                                                                                                                                                                                                                           |
| <a id="id"></a> `id`                         | `string`                                                                                                       | The unique identifier for the invitation.                                                                                                                                                                                                                                                                                               |
| <a id="organizationid"></a> `organizationId` | `string`                                                                                                       | The ID of the Organization that the invitation is for.                                                                                                                                                                                                                                                                                  |
| <a id="publicmetadata"></a> `publicMetadata` | [OrganizationInvitationPublicMetadata](/docs/reference/types/metadata#organization-invitation-public-metadata) | Metadata that can be read from both the [Frontend API](/docs/reference/frontend-api){{ target: '_blank' }} and [Backend API](/docs/reference/backend-api){{ target: '_blank' }}, but can be set only from the Backend API. Once the user accepts the invitation and signs up, these metadata will end up in the user's public metadata. |
| <a id="revoke"></a> `revoke`                 | <code>() => Promise\<OrganizationInvitationResource\></code>                                                   | Revokes the invitation so it can no longer be accepted.                                                                                                                                                                                                                                                                                 |
| <a id="role"></a> `role`                     | `string`                                                                                                       | The [Role](/docs/guides/organizations/control-access/roles-and-permissions) that the invited user will be assigned once they accept the invitation.                                                                                                                                                                                     |
| <a id="rolename"></a> `roleName`             | `string`                                                                                                       | The name of the [Role](/docs/guides/organizations/control-access/roles-and-permissions) that the invited user will be assigned.                                                                                                                                                                                                         |
| <a id="status"></a> `status`                 | <code>"expired" \| "pending" \| "accepted" \| "revoked"</code>                                                 | The current status of the invitation.                                                                                                                                                                                                                                                                                                   |
| <a id="updatedat"></a> `updatedAt`           | `Date`                                                                                                         | The date when the invitation was last updated.                                                                                                                                                                                                                                                                                          |
