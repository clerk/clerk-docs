# `PhoneNumber` Verification

These are all methods on [the `PhoneNumber` class](/reference-js/phone-number/phone-number) that allow you to verify a user's phone number.

## `prepareVerification()`

```typescript
function prepareVerification(): Promise<PhoneNumber>;
```

Kick off the verification process for this phone number. An SMS message with a one-time code will be sent to the phone number value.

## `attemptVerification()`

```typescript
function attemptVerification(params: AttemptPhoneNumberVerificationParams): Promise<PhoneNumber>;
```

Attempts to verify this phone number, passing the one-time code that was sent as an SMS message. The code will be sent when calling the [PhoneNumber.prepareVerification()](https://clerk.com/docs/reference/clerkjs/phonenumber#prepare-verification) method.
