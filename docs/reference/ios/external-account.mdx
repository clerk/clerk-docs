---
title: '`ExternalAccount`'
description: Manage connected OAuth accounts for a user.
sdk: ios
---

`ExternalAccount` represents a connected OAuth identity such as Google or GitHub.

## Add an external account

```swift
@Environment(Clerk.self) private var clerk

if let user = clerk.user {
  let account = try await user.createExternalAccount(provider: .google)
  print(account.provider)
}
```

## Reauthorize an account

```swift
let refreshed = try await account.reauthorize()
```

## Remove an account

```swift
_ = try await account.destroy()
```

## Key properties

<Properties>
  - `provider`
  - `String`

  Provider name (e.g., `google`).

  ---

  - `providerUserId`
  - `String`

  User identifier at the provider.

  ---

  - `emailAddress`
  - `String`

  Provider email address.

  ---

  - `username`
  - `String?`

  Provider username, if available.

  ---

  - `verification`
  - `Verification?`

  Verification state.
</Properties>
