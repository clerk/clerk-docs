Navigation options used to replace or push history changes.
Both `routerPush` & `routerReplace` OR none options should be passed.

| Name             | Type      | Description                                                                                     |
| ---------------- | --------- | ----------------------------------------------------------------------------------------------- |
| `routerDebug?`   | `boolean` | If `true`, the router will log debug information to the console.                                |
| `routerPush?`    | `never`   | A function which takes the destination path as an argument and performs a "push" navigation.    |
| `routerReplace?` | `never`   | A function which takes the destination path as an argument and performs a "replace" navigation. |

| Name              | Type                                                                                                | Description                                                                                     |
| ----------------- | --------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------- |
| `routerDebug?`    | `boolean`                                                                                           | If `true`, the router will log debug information to the console.                                |
| `routerPush()`    | <code>(to: string, metadata?: \{ windowNavigate: (to: URL \| string) => void; \}) => unknown</code> | A function which takes the destination path as an argument and performs a "push" navigation.    |
| `routerReplace()` | <code>(to: string, metadata?: \{ windowNavigate: (to: URL \| string) => void; \}) => unknown</code> | A function which takes the destination path as an argument and performs a "replace" navigation. |
