# `getPhoneNumber()`

Retrieves a single [`PhoneNumber`](/docs/references/javascript/phone-number).

```ts
function getPhoneNumber(phoneNumberId: string): Promise<PhoneNumber>
```

## Parameters

<Properties>
  - `phoneNumberId`
  - `string`

  The ID of the phone number to retrieve.
</Properties>

## Example

```tsx
const phoneNumberId = 'idn_123'

const response = await clerkClient.phoneNumbers.getPhoneNumber(phoneNumberId)

console.log(response)
/*
_PhoneNumber {
  id: 'idn_123',
  phoneNumber: '15551234567',

```

## Backend API (BAPI) endpoint

This method in the SDK is a wrapper around the BAPI endpoint `GET/phone_numbers/{phone_number_id}`. See the [BAPI reference](/docs/reference/backend-api/tag/Phone-Numbers#operation/GetPhoneNumber){{ target: '_blank' }} for more details.
