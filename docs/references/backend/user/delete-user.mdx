# `deleteUser()`

Deletes a user given a valid ID. Throws an error otherwise.

```tsx
const userId = 'my-user-id';

const user = await clerkClient.users.deleteUser(userId);
```

## Required parameters

| Name | Type | Description |
| --- | --- | --- |
| `userId` | `string` | The ID of the user to delete. |
