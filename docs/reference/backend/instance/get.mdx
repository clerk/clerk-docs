## Backend API (BAPI) endpoint

This method in the SDK is a wrapper around the BAPI endpoint `GET/instance`. See the [BAPI reference](/docs/reference/backend-api/tag/instance-settings/GET/instance){{ target: '_blank' }} for more information.
