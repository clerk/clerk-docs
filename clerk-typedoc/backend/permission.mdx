The Backend `Permission` object represents an organization permission that can be assigned to organization roles.

## Properties

| Property                               | Type     | Description                                                           |
| -------------------------------------- | -------- | --------------------------------------------------------------------- |
| <a id="createdat"></a> `createdAt`     | `number` | The date when the permission was first created.                       |
| <a id="description"></a> `description` | `string` | A description of the permission.                                      |
| <a id="id"></a> `id`                   | `string` | The unique identifier for the permission.                             |
| <a id="key"></a> `key`                 | `string` | The unique key of the permission, in the format `org:feature:action`. |
| <a id="name"></a> `name`               | `string` | The name of the permission.                                           |
| <a id="updatedat"></a> `updatedAt`     | `number` | The date when the permission was last updated.                        |
