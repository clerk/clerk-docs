This hook allows you to access the Signal-based `SignUp` resource.

## Returns

| Name          | Type                                                             | Description                                                                               |
| ------------- | ---------------------------------------------------------------- | ----------------------------------------------------------------------------------------- |
| `errors`      | [`SignUpErrors`](/docs/reference/javascript/types/errors)        | The errors that occurred during the last fetch of the underlying `SignUpFuture` resource. |
| `fetchStatus` | <code>"idle" \| "fetching"</code>                                | The fetch status of the underlying `SignUpFuture` resource.                               |
| `signUp`      | [`SignUpFutureResource`](/docs/reference/objects/sign-up-future) | The underlying `SignUpFuture` resource.                                                   |

## Example

```ts
import { useSignUp } from "@clerk/react";

function SignUpForm() {
  const { signUp, errors, fetchStatus } = useSignUp();
  //
}
```
