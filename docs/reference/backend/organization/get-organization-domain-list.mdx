## Usage

<Include src="_partials/backend/usage" />

```tsx
const organizationId = 'org_123'

const response = await clerkClient.organizations.getOrganizationDomainList({ organizationId })
```

## Backend API (BAPI) endpoint

This method in the SDK is a wrapper around the BAPI endpoint `GET/organizations/{organization_id}/domains`. See the [BAPI reference](/docs/reference/backend-api/tag/organization-domains/GET/organizations/%7Borganization_id%7D/domains){{ target: '_blank' }} for more information.
