The `BillingPlanPrice` type represents a specific possible price for a given Clerk Billing Plan.

## Properties

| Property                                         | Type                                                                                  | Description                                                                            |
| ------------------------------------------------ | ------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------- |
| <a id="annualmonthlyfee"></a> `annualMonthlyFee` | <code>null \| [BillingMoneyAmount](/docs/reference/types/billing-money-amount)</code> | The effective monthly price when billed annually or `null` if the price is not annual. |
| <a id="fee"></a> `fee`                           | <code>null \| [BillingMoneyAmount](/docs/reference/types/billing-money-amount)</code> | The monthly price or `null` if the price is not monthly.                               |
| <a id="id"></a> `id`                             | `string`                                                                              | The unique identifier of the price                                                     |
| <a id="isdefault"></a> `isDefault`               | `boolean`                                                                             | Whether this price is the default price for the plan it's associated with.             |
| <a id="unitprices"></a> `unitPrices?`            | <code>[BillingPlanUnitPrice](/docs/reference/types/billing-plan-unit-price)[]</code>  | The individual unit prices applicable to this price.                                   |
