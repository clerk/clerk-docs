<Properties>
  - `enabled`
  - `boolean | ((url: URL) => boolean)`

  Enable Frontend API proxy handling. When `true`, requests to the proxy path are forwarded to [Clerk's Frontend API](/docs/reference/frontend-api){{ target: '_blank' }}, and the `proxyUrl` is automatically derived for authentication handshake.

  ---

  - `path?`
  - `string`

  The path prefix for proxy requests. Defaults to `'/__clerk'`. Must be unique and not conflict with other routes in your application.
</Properties>
