Refreshes the given session.

Returns the refreshed token.

```typescript
function refreshSession(sessionId: string, params: { expired_token: string; format?: "cookie" | "token"; refresh_token: string; request_headers?: Record<string, string[]>; request_origin: string; request_originating_ip?: string; suffixed_cookies?: boolean } & { format: "token" }): Promise<Token>
```

## Parameters


| Parameter | Type | Description |
| ------ | ------ | ------ |
| `sessionId` | `string` | The ID of the session to refresh. |
| `params` | <code>\{ expired_token: string; format?: "cookie" \| "token"; refresh_token: string; request_headers?: Record\<string, string[]\>; request_origin: string; request_originating_ip?: string; suffixed_cookies?: boolean; \}</code> & <code>\{ format: "token"; \}</code> | The parameters to refresh the session. |
| `params.expired_token` | `string` | The expired token to refresh. |
| `params.format` | `"token"` | The format of the token to refresh. |
| `params.refresh_token` | `string` | The refresh token to refresh. |
| `params.request_headers?` | `Record`\<`string`, <code>string[]</code>\> | The headers of the request. |
| `params.request_origin` | `string` | The origin of the request. |
| `params.request_originating_ip?` | `string` | The originating IP address of the request. |
| `params.suffixed_cookies?` | `boolean` | Whether to use suffixed cookies. |
