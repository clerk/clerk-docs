The `<OrganizationList />` component is used to display organization related memberships, invitations, and suggestions for the user.

### Force Organizations

For specific applications, we want to prohibit people from using their personal accounts as they need to be part of an organization.
Use <code>{"<OrganizationList/>"}</code> with <code>{"hidePeronsal"}</code> in order to guide your user to join or create an organization in order to continue.
```tsx copy filename="organization-portal.tsx"
export default function OrganizationListPage() {
  return (
    <OrganizationList
        hidePersonal={true}
        {...}
    />
  );
};
```
