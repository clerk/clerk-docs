Updates an [Organization](/docs/reference/backend/types/backend-organization).

Returns the updated [Organization](/docs/reference/backend/types/backend-organization).

```typescript
function updateOrganization(organizationId: string, params: { adminDeleteEnabled?: boolean; maxAllowedMemberships?: number; name?: string; privateMetadata?: OrganizationPrivateMetadata; publicMetadata?: OrganizationPublicMetadata; slug?: string }): Promise<Organization>
```

## Parameters


| Parameter | Type | Description |
| ------ | ------ | ------ |
| `organizationId` | `string` | The ID of the Organization to update. |
| `params` | <code>\{ adminDeleteEnabled?: boolean; maxAllowedMemberships?: number; name?: string; privateMetadata?: [OrganizationPrivateMetadata](/docs/reference/types/metadata#organization-private-metadata); publicMetadata?: [OrganizationPublicMetadata](/docs/reference/types/metadata#organization-public-metadata); slug?: string; \}</code> | The parameters to update the Organization with. |
| `params.adminDeleteEnabled?` | `boolean` | Whether the Organization allows admins to delete users. |
| `params.maxAllowedMemberships?` | `number` | The maximum number of memberships allowed in the Organization. `0` means unlimited. |
| `params.name?` | `string` | The name to update the Organization with. |
| `params.privateMetadata?` | [OrganizationPrivateMetadata](/docs/reference/types/metadata#organization-private-metadata) | **Deprecated.** Updating metadata via [`updateOrganization()`](/docs/reference/backend/organization/update-organization) is deprecated. Use [`updateOrganizationMetadata()`](/docs/reference/backend/organization/update-organization-metadata) for partial updates (deep merge) or [`replaceOrganizationMetadata()`](/docs/reference/backend/organization/replace-organization-metadata) for full replacement. |
| `params.publicMetadata?` | [OrganizationPublicMetadata](/docs/reference/types/metadata#organization-public-metadata) | **Deprecated.** Updating metadata via [`updateOrganization()`](/docs/reference/backend/organization/update-organization) is deprecated. Use [`updateOrganizationMetadata()`](/docs/reference/backend/organization/update-organization-metadata) for partial updates (deep merge) or [`replaceOrganizationMetadata()`](/docs/reference/backend/organization/replace-organization-metadata) for full replacement. |
| `params.slug?` | `string` | The slug to update the Organization with. |
