The `BillingPaymentMethodResource` type represents a payment method for a checkout session.

## Properties

| Property                               | Type                                                                                                             | Description                                                                                                                                                                                            |
| -------------------------------------- | ---------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| <a id="cardtype"></a> `cardType`       | `string`                                                                                                         | The brand or type of card. For example, `'visa'` or `'mastercard'`.                                                                                                                                    |
| <a id="id"></a> `id`                   | `string`                                                                                                         | The unique identifier for the payment method.                                                                                                                                                          |
| <a id="isdefault"></a> `isDefault`     | `boolean`                                                                                                        | Whether the payment method is set as the default for the account.                                                                                                                                      |
| <a id="isremovable"></a> `isRemovable` | `boolean`                                                                                                        | Whether the payment method can be removed by the user.                                                                                                                                                 |
| <a id="last4"></a> `last4`             | `string`                                                                                                         | The last four digits of the payment method.                                                                                                                                                            |
| <a id="makedefault"></a> `makeDefault` | <code>(params?: \{ orgId?: string; \}) => Promise\<null\></code>                                                 | A function that sets this payment method as the default for the account. Accepts the following parameters: <ul> <li>`orgId?` (`string`): The ID of the organization to set as the default.</li> </ul>  |
| <a id="pathroot"></a> `pathRoot`       | `string`                                                                                                         | The root path of the resource.                                                                                                                                                                         |
| <a id="paymenttype"></a> `paymentType` | <code>"link" \| "card"</code>                                                                                    | The type of payment method. For example, `'card'` or `'link'`.                                                                                                                                         |
| <a id="remove"></a> `remove`           | <code>(params?: \{ orgId?: string; \}) => Promise\<[DeletedObjectResource](deleted-object-resource.mdx)\></code> | A function that removes this payment method from the account. Accepts the following parameters: <ul> <li>`orgId?` (`string`): The ID of the organization to remove the payment method from.</li> </ul> |
| <a id="status"></a> `status`           | <code>"active" \| "disconnected" \| "expired"</code>                                                             | The current status of the payment method.                                                                                                                                                              |
| <a id="wallettype"></a> `walletType`   | <code>undefined \| string</code>                                                                                 | The type of digital wallet, if applicable. For example, `'apple_pay'`, or `'google_pay'`.                                                                                                              |
