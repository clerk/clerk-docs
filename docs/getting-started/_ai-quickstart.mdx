          <Tab>
            ### Connect Claude to Clerk's MCP server

            Run the following command to add Clerk's MCP server:

            ```bash
            claude mcp add clerk --transport http https://mcp.clerk.com/mcp
            ```

            1. Run `/mcp` once you've opened a Claude Code session to verify the connection.
            1. Ask Claude to use Clerk MCP tools when you want up-to-date auth snippets and Next.js implementation patterns.

            [Learn more about the Clerk MCP](/docs/guides/ai/mcp/clerk-mcp-server)
          </Tab>

          <Tab>
            ### Connect Cursor to Clerk's MCP server

            Add the following configuration to your MCP settings JSON:

            ```json
            {
              "mcpServers": {
                "clerk": {
                  "url": "https://mcp.clerk.com/mcp"
                }
              }
            }
            ```

            1. Open **Cursor Settings**.
            1. Go to **Tools & MCP**.
            1. Add a new MCP server named **Clerk** and save the configuration.

            [Learn more about the Clerk MCP](/docs/guides/ai/mcp/clerk-mcp-server)
          </Tab>

          <Tab>
            ### Connect VSCode to Clerk's MCP server

            Use the MCP server command below when VS Code prompts for configuration:

            ```bash
            npx mcp-remote https://mcp.clerk.com/mcp
            ```

            1. Open the Command Palette with `Cmd/Ctrl + Shift + P`.
            1. Run **MCP: Add Server**.
            1. Enter **Clerk** as the server name.

            [Learn more about the Clerk MCP](/docs/guides/ai/mcp/clerk-mcp-server)
          </Tab>

          <Tab>
            ### Connect Windsurf to Clerk's MCP server

            Add the following configuration to your MCP settings JSON:

            ```json
            {
              "mcpServers": {
                "clerk": {
                  "command": "npx",
                  "args": ["-y", "mcp-remote", "https://mcp.clerk.com/mcp"]
                }
              }
            }
            ```

            1. Open Windsurf settings.
            1. Scroll to **Cascade -> MCP Servers**.
            1. Paste the Clerk configuration into your MCP settings file.

            [Learn more about the Clerk MCP](/docs/guides/ai/mcp/clerk-mcp-server)
          </Tab>
