The `DeletedObject` class represents an item that has been deleted from the database.

## Properties

| Name | Type | Description |
| --- | --- | --- |
| `object` | `string` | The object type that has been deleted. |
| `id` | `string \| null` | The ID of the deleted item. |
| `slug` | `string \| null` | The slug of the deleted item. |
| `deleted` | `boolean` | A boolean checking if the item has been deleted or not. |
