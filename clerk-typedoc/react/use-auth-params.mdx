## Parameters

| Parameter  | Type                                                          | Description                                                                                                                                                                  |
| ---------- | ------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `options?` | <code>null \| \{ treatPendingAsSignedOut?: boolean; \}</code> | An object containing options for the `useAuth()` hook. `treatPendingAsSignedOut` indicates whether pending sessions are considered as signed out or not. Defaults to `true`. |
