### `mountSignUp()`

Mounts a sign up flow component at the target element.

```typescript
function mountSignUp(targetNode: HTMLDivElement, signUpProps?: SignUpProps): void
```

#### Parameters


| Parameter | Type | Description |
| ------ | ------ | ------ |
| `targetNode` | `HTMLDivElement` | Target node to mount the SignUp component. |
| `signUpProps?` | [`SignUpProps`](../sign-up-props.mdx) | sign up configuration parameters. |
