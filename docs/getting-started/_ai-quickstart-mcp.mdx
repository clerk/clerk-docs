<Steps>
  ### Select and connect your agent to Clerk's MCP Server

  <Tabs items={["Cursor", "Claude", "VSCode", "Windsurf", "Zed", "Codex", "Other clients"]}>
    <Tab>
      ### Connecting Cursor to Clerk's MCP server

      Add the following configuration to your MCP settings JSON:

      ```json
      {
        "mcpServers": {
          "clerk": {
            "url": "https://mcp.clerk.com/mcp"
          }
        }
      }
      ```

      1. Open **Cursor Settings**.
      1. Go to **Tools & MCP**.
      1. Add a new MCP server named **Clerk** and save the configuration.

      [Learn more about the Clerk MCP](/docs/guides/ai/mcp/clerk-mcp-server)
    </Tab>

    <Tab>
      ### Connecting Claude to Clerk's MCP server

      <Tabs items={["Claude Code", "Claude.ai", "Claude Desktop"]}>
        <Tab>
          Run the following command to add Clerk's MCP server:

          ```bash
          claude mcp add clerk --transport http https://mcp.clerk.com/mcp
          ```

          1. Run `/mcp` once you've opened a Claude Code session to verify the connection.
          1. Ask Claude to use Clerk MCP tools when you want up-to-date auth snippets and Next.js implementation patterns.

          [Learn more about the Clerk MCP](/docs/guides/ai/mcp/clerk-mcp-server)
        </Tab>

        <Tab>
          1. Open a conversation on [claude.ai](https://claude.ai).
          1. Click the **MCP integrations** icon in the input bar.
          1. Add the Clerk MCP server URL:

          ```
          https://mcp.clerk.com/mcp
          ```

          [Learn more about the Clerk MCP](/docs/guides/ai/mcp/clerk-mcp-server)
        </Tab>

        <Tab>
          Add the following to your Claude Desktop configuration file (`claude_desktop_config.json`):

          ```json
          {
            "mcpServers": {
              "clerk": {
                "command": "npx",
                "args": ["-y", "mcp-remote", "https://mcp.clerk.com/mcp"]
              }
            }
          }
          ```

          1. Open Claude Desktop settings.
          1. Navigate to the **Developer** section.
          1. Add the Clerk MCP server configuration and restart Claude Desktop.

          [Learn more about the Clerk MCP](/docs/guides/ai/mcp/clerk-mcp-server)
        </Tab>
      </Tabs>
    </Tab>

    <Tab>
      ### Connecting VS Code to Clerk's MCP server

      Use the MCP server command below when VS Code prompts for configuration:

      ```bash
      npx mcp-remote https://mcp.clerk.com/mcp
      ```

      1. Open the Command Palette with `Cmd/Ctrl + Shift + P`.
      1. Run **MCP: Add Server**.
      1. Enter **Clerk** as the server name.

      [Learn more about the Clerk MCP](/docs/guides/ai/mcp/clerk-mcp-server)
    </Tab>

    <Tab>
      ### Connecting Windsurf to Clerk's MCP server

      Add the following configuration to your MCP settings JSON:

      ```json
      {
        "mcpServers": {
          "clerk": {
            "command": "npx",
            "args": ["-y", "mcp-remote", "https://mcp.clerk.com/mcp"]
          }
        }
      }
      ```

      1. Open Windsurf settings.
      1. Scroll to **Cascade -> MCP Servers**.
      1. Paste the Clerk configuration into your MCP settings file.

      [Learn more about the Clerk MCP](/docs/guides/ai/mcp/clerk-mcp-server)
    </Tab>

    <Tab>
      ### Connecting Zed to Clerk's MCP server

      Add the following to your Zed settings (`settings.json`):

      ```json
      {
        "context_servers": {
          "clerk": {
            "command": {
              "path": "npx",
              "args": ["-y", "mcp-remote", "https://mcp.clerk.com/mcp"]
            }
          }
        }
      }
      ```

      [Learn more about the Clerk MCP](/docs/guides/ai/mcp/clerk-mcp-server)
    </Tab>

    <Tab>
      ### Connecting Codex to Clerk's MCP server

      Pass the Clerk MCP server when starting a Codex session:

      ```bash
      codex --mcp-config '{"clerk":{"url":"https://mcp.clerk.com/mcp"}}'
      ```

      [Learn more about the Clerk MCP](/docs/guides/ai/mcp/clerk-mcp-server)
    </Tab>

    <Tab>
      ### Connecting to Clerk's MCP server

      Most MCP-compatible clients can connect using the Clerk MCP server URL:

      ```
      https://mcp.clerk.com/mcp
      ```

      If your client requires a command-based configuration, use:

      ```json
      {
        "mcpServers": {
          "clerk": {
            "command": "npx",
            "args": ["-y", "mcp-remote", "https://mcp.clerk.com/mcp"]
          }
        }
      }
      ```

      [Learn more about the Clerk MCP](/docs/guides/ai/mcp/clerk-mcp-server)
    </Tab>
  </Tabs>

  ### Configure application and create your Clerk account

  This prompt quickstart auto-generates temporary API keys so you can test Clerk right away.

  Claim your keys with the callout in your app, which will allow you to create or log in to Clerk. Or manually [sign up](https://dashboard.clerk.com/sign-up) to link your keys, customize your auth settings, and view more features.
