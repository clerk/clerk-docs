The `BillingPaymentTotals` type represents the per-payment cost breakdown, including any base fee
and per-unit (e.g., seats) subtotals.

## Properties

| Property                                    | Type                                                                                  | Description                                                                                                  |
| ------------------------------------------- | ------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------ |
| <a id="basefee"></a> `baseFee?`             | <code>null \| [BillingMoneyAmount](/docs/reference/types/billing-money-amount)</code> | The flat base fee charged on top of any per-unit fees.                                                       |
| <a id="discounts"></a> `discounts?`         | <code>null \| [BillingDiscounts](billing-discounts.mdx)</code>                        | Discounts applied to this payment such as mid-cycle prorated seat discounts. `null` when no discounts apply. |
| <a id="grandtotal"></a> `grandTotal`        | [`BillingMoneyAmount`](/docs/reference/types/billing-money-amount)                    | The total amount for the payment, including taxes and after credits/discounts are applied.                   |
| <a id="perunittotals"></a> `perUnitTotals?` | <code>[BillingPerUnitTotal](/docs/reference/types/billing-per-unit-total)[]</code>    | Per-unit cost breakdown for this payment (e.g., seats).                                                      |
| <a id="subtotal"></a> `subtotal`            | [`BillingMoneyAmount`](/docs/reference/types/billing-money-amount)                    | The price of the items before taxes, credits, or discounts are applied.                                      |
| <a id="taxtotal"></a> `taxTotal`            | [`BillingMoneyAmount`](/docs/reference/types/billing-money-amount)                    | The amount of tax included in the payment.                                                                   |
