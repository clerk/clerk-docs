# `<CreateOrganization />`

---

<Images alt="Organization Switcher Example" width={992} height={568} src="/images/ui-components/component-org_switcher.svg"/>
The `<CreateOrganization/>` component is used to render an organization creation UI that allows users to create brand new organizations within you application.

## Usage

```jsx copy filename="/pages/create-organization/[[...index]].tsx"
import { CreateOrganization } from "@clerk/nextjs";

export default function CreateOrganizationPage() {
  return <CreateOrganization />;
}
```
