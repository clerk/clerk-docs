### `mountUserButton()`

Mount a user button component at the target element.

```typescript
function mountUserButton(targetNode: HTMLDivElement, userButtonProps?: UserButtonProps): void
```

#### Parameters


| Parameter | Type | Description |
| ------ | ------ | ------ |
| `targetNode` | `HTMLDivElement` | Target node to mount the UserButton component. |
| `userButtonProps?` | `UserButtonProps` | User button configuration parameters. |
