## Usage

<Include src="_partials/backend/usage" />

```tsx
const phoneNumberId = 'idn_123'

const response = await clerkClient.phoneNumbers.getPhoneNumber(phoneNumberId)
```

## Backend API (BAPI) endpoint

This method in the SDK is a wrapper around the BAPI endpoint `GET/phone_numbers/{phone_number_id}`. See the [BAPI reference](/docs/reference/backend-api/tag/phone-numbers/GET/phone_numbers/\{phone_number_id}){{ target: '_blank' }} for more information.
