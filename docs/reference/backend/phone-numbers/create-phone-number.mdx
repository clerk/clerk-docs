## Usage

<Include src="_partials/backend/usage" />

```tsx
const response = await clerkClient.phoneNumbers.createPhoneNumber({
  userId: 'user_123',
  phoneNumber: '15551234567',
  primary: true,
  verified: true,
})
```

## Backend API (BAPI) endpoint

This method in the SDK is a wrapper around the BAPI endpoint `POST/phone_numbers`. See the [BAPI reference](/docs/reference/backend-api/tag/phone-numbers/POST/phone_numbers){{ target: '_blank' }} for more information.
