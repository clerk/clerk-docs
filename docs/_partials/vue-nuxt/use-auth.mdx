The [`useAuth()`](/docs/reference/composables/use-auth) composable provides access to the current user's authentication state. It includes the `isSignedIn` property to check if the active user is signed in, which is helpful for protecting a page.

```vue {{ filename: 'app/pages/protected-page.vue' }}
<script setup>
const { isSignedIn, isLoaded, userId } = useAuth()
</script>

<template>
  <!-- Use `isLoaded` to check if Clerk is loaded -->
  <div v-if="!isLoaded">Loading...</div>
  <!-- Use `isSignedIn` to check if the user is signed in -->
  <div v-else-if="!isSignedIn">Sign in to access this page</div>
  <!-- Use `userId` to access the current user's ID -->
  <div v-else>Hello, {{ userId }}!</div>
</template>
```
