Holds the verification details of an Organization's [Verified Domain](/docs/guides/organizations/add-members/verified-domains).

## Properties

| Property                           | Type                                    | Description                                                      |
| ---------------------------------- | --------------------------------------- | ---------------------------------------------------------------- |
| <a id="attempts"></a> `attempts`   | `number`                                | The number of verification attempts that have been made.         |
| <a id="expiresat"></a> `expiresAt` | `Date`                                  | The date and time when the current verification attempt expires. |
| <a id="status"></a> `status`       | <code>"unverified" \| "verified"</code> | The current status of the domain verification.                   |
| <a id="strategy"></a> `strategy`   | `"email_code"`                          | The strategy used to verify the domain.                          |
