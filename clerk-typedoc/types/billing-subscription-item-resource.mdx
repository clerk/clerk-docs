The `BillingSubscriptionItemResource` type represents an item in a subscription.

## Properties

| Property                                       | Type                                                                                                                                         | Description                                                                                                                                                                             |
| ---------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| <a id="amount"></a> `amount?`                  | [`BillingMoneyAmount`](/docs/reference/javascript/types/billing-money-amount)                                                                | The amount charged for the subscription item.                                                                                                                                           |
| <a id="cancel"></a> `cancel`                   | <code>(params: \{ orgId?: string; \}) => Promise\<[DeletedObjectResource](/docs/reference/javascript/types/deleted-object-resource)\></code> | A function to cancel the subscription item. Accepts the following parameters: <ul> <li>`orgId?` (`string`): The ID of the organization to cancel the subscription item from.</li> </ul> |
| <a id="canceledat"></a> `canceledAt`           | <code>null \| Date</code>                                                                                                                    | The date and time when the subscription item was canceled. `null` if the subscription item is not canceled.                                                                             |
| <a id="createdat"></a> `createdAt`             | `Date`                                                                                                                                       | The date and time when the subscription item was created.                                                                                                                               |
| <a id="credit"></a> `credit?`                  | <code>\{ amount: [BillingMoneyAmount](/docs/reference/javascript/types/billing-money-amount); \}</code>                                      | The credit from a previous purchase that is being applied to the subscription item.                                                                                                     |
| `credit.amount`                                | [`BillingMoneyAmount`](/docs/reference/javascript/types/billing-money-amount)                                                                | The amount of credit from a previous purchase that is being applied to the subscription item.                                                                                           |
| <a id="id"></a> `id`                           | `string`                                                                                                                                     | The unique identifier for the subscription item.                                                                                                                                        |
| <a id="isfreetrial"></a> `isFreeTrial`         | `boolean`                                                                                                                                    | Whether the subscription item is for a free trial.                                                                                                                                      |
| <a id="pastdueat"></a> `pastDueAt`             | <code>null \| Date</code>                                                                                                                    | The date and time when the subscription item became past due. `null` if the subscription item is not past due.                                                                          |
| <a id="pathroot"></a> `pathRoot`               | `string`                                                                                                                                     | The root path of the resource.                                                                                                                                                          |
| <a id="paymentmethodid"></a> `paymentMethodId` | `string`                                                                                                                                     | The unique identifier for the payment method being used for the subscription item.                                                                                                      |
| <a id="periodend"></a> `periodEnd`             | <code>null \| Date</code>                                                                                                                    | The date and time when the current billing period ends. `null` if not set.                                                                                                              |
| <a id="periodstart"></a> `periodStart`         | `Date`                                                                                                                                       | The date and time when the current billing period starts.                                                                                                                               |
| <a id="plan"></a> `plan`                       | [`BillingPlanResource`](/docs/reference/javascript/types/billing-plan-resource)                                                              | The plan associated with the subscription item.                                                                                                                                         |
| <a id="planperiod"></a> `planPeriod`           | <code>"month" \| "annual"</code>                                                                                                             | The billing period for the subscription item.                                                                                                                                           |
| <a id="status"></a> `status`                   | <code>"active" \| "ended" \| "upcoming" \| "past_due"</code>                                                                                 | The status of the subscription item.                                                                                                                                                    |
