> [!WARNING]
> This composable should only be used for advanced use cases, such as building a completely custom OAuth flow or as an escape hatch to access to the `Clerk` object.

The `useClerk()` composable provides access to the [`Clerk`](/docs/reference/objects/clerk) object, giving you the ability to build alternatives to any [Clerk component](/docs/reference/components/overview).

## Returns

The `useClerk()` composable returns the `Clerk` object, which includes all the methods and properties listed in the [`Clerk` reference](/docs/reference/objects/clerk).
