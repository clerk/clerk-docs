The Backend `Role` object represents an organization role that can be assigned to organization members.

## Properties

| Property                                           | Type                                        | Description                                                       |
| -------------------------------------------------- | ------------------------------------------- | ----------------------------------------------------------------- |
| <a id="createdat"></a> `createdAt`                 | `number`                                    | The date when the role was first created.                         |
| <a id="description"></a> `description`             | <code>null \| string</code>                 | A description of the role.                                        |
| <a id="id"></a> `id`                               | `string`                                    | The unique identifier for the role.                               |
| <a id="iscreatoreligible"></a> `isCreatorEligible` | `boolean`                                   | Whether this role is eligible to be an organization creator role. |
| <a id="key"></a> `key`                             | `string`                                    | The unique key of the role, in the format `org:role`.             |
| <a id="name"></a> `name`                           | `string`                                    | The name of the role.                                             |
| <a id="permissions"></a> `permissions`             | <code>[Permission](permission.mdx)[]</code> | The permissions assigned to the role.                             |
| <a id="updatedat"></a> `updatedAt`                 | `number`                                    | The date when the role was last updated.                          |
