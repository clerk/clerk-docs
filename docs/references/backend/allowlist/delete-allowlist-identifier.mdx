title: '`deleteAllowlistIdentifier()`'
description: Deletes an allowlist identifier, given a valid ID. Throws an error if the ID is invalid.
---

Deletes an allowlist identifier, given a valid ID. Throws an error if the ID is invalid.

```tsx
const allowlistIdentifierId = 'alid_randomid';

await clerkClient.allowlistIdentifiers.deleteAllowlistIdentifier(allowlistIdentifierId);
```

## Required parameters

| Name | Type | Description |
| ---- | --- | --- |
| `allowlistIdentifierId` | `string` | The ID of the allowlist identifier to delete. |
