The `<UserProfileView />` component renders a complete native profile management interface using SwiftUI on iOS and Jetpack Compose on Android. It allows users to manage their account settings, security options, and connected accounts.

## Usage

The following example shows how to use the `<UserProfileView />` in your Expo app.

### Basic usage

```tsx {{ filename: 'app/(home)/profile.tsx' }}
import { UserProfileView } from '@clerk/expo/native'
import { useRouter } from 'expo-router'

export default function ProfileScreen() {
  const router = useRouter()

  return <UserProfileView onSignOut={() => router.replace('/(auth)/sign-in')} />
}
```

### Full screen profile view

```tsx {{ filename: 'app/(home)/profile.tsx' }}
import { View } from 'react-native'
import { UserProfileView } from '@clerk/expo/native'
import { useRouter } from 'expo-router'

export default function ProfileScreen() {
  const router = useRouter()

  return (
    <View style={{ flex: 1 }}>
      <UserProfileView isDismissable={false} onSignOut={() => router.replace('/(auth)/sign-in')} />
    </View>
  )
}
```

### As a modal

```tsx {{ filename: 'app/(home)/index.tsx' }}
import { useState } from 'react'
import { View, Button, Modal } from 'react-native'
import { UserProfileView } from '@clerk/expo/native'
import { useRouter } from 'expo-router'

export default function HomeScreen() {
  const [showProfile, setShowProfile] = useState(false)
  const router = useRouter()

  return (
    <View style={{ flex: 1 }}>
      <Button title="Open Profile" onPress={() => setShowProfile(true)} />

      <Modal visible={showProfile} animationType="slide">
        <UserProfileView
          isDismissable={true}
          onSignOut={() => {
            setShowProfile(false)
            router.replace('/(auth)/sign-in')
          }}
        />
      </Modal>
    </View>
  )
}
```

## Properties

<Properties>
  - `presentation`
  - `'modal' | 'inline'`

  How the profile view is presented. Defaults to `'modal'`.

  - `'modal'` - Presents a full-screen native modal
  - `'inline'` - Renders directly within the React Native view hierarchy

  ---

  - `isDismissable`
  - `boolean`

  Whether the profile view can be dismissed by the user. When `true`, a dismiss button appears in the navigation bar and the view can be used in sheets or other dismissable contexts. When `false`, no dismiss button is shown, making it suitable for fullscreen usage. Defaults to `true`.

  ---

  - `onSignOut`
  - `() => void`

  Callback fired when the user signs out. Use this to navigate to your sign-in screen.

  ---

  - `onDismiss`
  - `() => void`

  Callback fired when the user dismisses the profile view. Only applies when `presentation` is `'inline'`.

  ---

  - `style`
  - `StyleProp<ViewStyle>`

  Style applied to the container view.
</Properties>

## Available features

The `<UserProfileView />` provides access to comprehensive account management:

### Profile management

- View and edit profile photo
- Update first and last name
- Change username (if enabled)

### Email addresses

- View all email addresses
- Add new email addresses
- Verify email addresses
- Set primary email
- Remove email addresses

### Phone numbers

- View all phone numbers
- Add new phone numbers
- Verify phone numbers via SMS
- Set primary phone
- Remove phone numbers

### Password & security

- Change password
- Set password (if none exists)
- Remove password (if other auth methods available)

### Multi-factor authentication

- Enable/disable SMS 2FA
- Set up authenticator app (TOTP)
- View and regenerate backup codes
- Remove MFA methods

### Passkeys

- View registered passkeys
- Add new passkeys
- Rename passkeys
- Remove passkeys

### Connected accounts

- View connected OAuth providers
- Connect new OAuth accounts (Google, Apple, GitHub, etc.)
- Disconnect OAuth accounts

### Active sessions

- View all active sessions/devices
- See current device details
- Sign out from other sessions
- Sign out from all sessions

### Account actions

- Sign out
- Delete account

## Session synchronization

Changes made in the `<UserProfileView />` are automatically synchronized with the JavaScript SDK. For example:

```tsx
import { UserProfileView } from '@clerk/expo/native'
import { useUser } from '@clerk/expo'

function ProfileScreen() {
  const { user } = useUser()

  // After user updates their name in UserProfileView,
  // this will automatically reflect the new name
  console.log(user?.firstName)

  return <UserProfileView />
}
```
