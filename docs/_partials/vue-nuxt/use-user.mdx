The [`useUser()`](/docs/reference/composables/use-user) composable provides access to the current user's [`User`](/docs/reference/objects/user) object, which contains the current user's data such as their ID. It also includes the `isSignedIn` property to check if the active user is signed in, which is helpful for protecting a page.

```vue {{ filename: 'GetCurrentUser.vue' }}
<script setup>
import { useUser } from '@clerk/vue'

const { isSignedIn, user, isLoaded } = useUser()
</script>

<template>
  <!-- Use `isLoaded` to check if Clerk is loaded -->
  <div v-if="!isLoaded">Loading...</div>

  <!-- Use `isSignedIn` to check if the user is signed in -->
  <div v-else-if="!isSignedIn">Sign in to view this page</div>

  <div v-else>Hello {{ user?.id }}!</div>
</template>
```
