### `getDomain()`

Gets a domain for an Organization based on the given domain ID.

Returns an [`OrganizationDomainResource`](/docs/reference/types/organization-domain-resource) object.

> [!WARNING]
> You must have [**Verified domains**](/docs/guides/organizations/add-members/verified-domains) enabled in your app's settings in the Clerk Dashboard.

```typescript
function getDomain(domainId: { domainId: string }): Promise<OrganizationDomainResource>
```

#### Parameters


| Parameter | Type | Description |
| ------ | ------ | ------ |
| `domainId` | <code>\{ domainId: string; \}</code> | The unique identifier of the domain to get. |
