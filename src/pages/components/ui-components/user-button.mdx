# `<UserButton />`

## Overview

The `<UserButton/>` component is used to render the familiar user button UI popularized by Google. Learn more about the `<UserButton/>` component, what it is, and how it is used by reading our [User Button component guide](https://clerk.dev/docs/users/user-button).

![](https://clerk.dev/_next/image?url=https%3A%2F%2Fcdn.sanity.io%2Fimages%2Fe1ql88v4%2Fproduction%2Ff45d86fae42cd6d564cddaa82cbf08127419f361-471x526.svg%3Ffit%3Dmax%26auto%3Dformat&w=1080&q=75)

## Embedding a `<UserButton />` component

```jsx copy filename="userButtonExample.[jsx/tsx]"
import {
  ClerkProvider,
  SignedIn,
  SignedOut,
  SignInButton,
  UserButton,
} from "@clerk/nextjs";

function Header() {
  return (
    <header
      style={{ display: "flex", justifyContent: "space-between", padding: 20 }}
    >
      <h1>My App</h1>
      <SignedIn>
        {/* Mount the UserButton component */}
        <UserButton />
      </SignedIn>
      <SignedOut>
        {/* Signed out users get sign in button */}
        <SignInButton />
      </SignedOut>
    </header>
  );
}

function MyApp({ pageProps }) {
  return (
    <ClerkProvider {...pageProps}>
      <Header />
    </ClerkProvider>
  );
}

export default MyApp;
```

## Custom Styling

The `<UserButton />` component can be highly customized through the [appearance prop](https://clerk.dev/docs/nextjs/appearance-prop) and can be styled using [CSS Modules](https://clerk.dev/docs/component-customization/using-css-modules), [Tailwind](https://clerk.dev/docs/component-customization/using-tailwind), [inline CSS objects](https://clerk.dev/docs/component-customization/using-inline-css), or [global CSS](https://clerk.dev/docs/component-customization/using-global-css).

## Customizing User Profile modal

By leveraging the appearance prop above, it's possible to customize the appearance of the `<UserProfile/>` component that is presented by clicking the `<UserButton/>`. Below is a simple example of how that can be achieved:

```js copy
<UserButton
  appearance={{
    userProfile: { elements: { breadcrumbs: "bg-slate-500" } },
  }}
/>
```

## Localization

The `<UserButton/>` component can be localized using the [localization prop](https://clerk.dev/docs/nextjs/localization). This allows you to customize the language used within the `<UserButton/> `component.
