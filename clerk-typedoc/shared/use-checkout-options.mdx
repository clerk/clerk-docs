| Property                             | Type                                  | Description                                                             |
| ------------------------------------ | ------------------------------------- | ----------------------------------------------------------------------- |
| <a id="for"></a> `for?`              | <code>"user" \| "organization"</code> | Specifies if the checkout is for an Organization. Defaults to `'user'`. |
