> [!IMPORTANT]
> If you're using React Router v7, enable the `v8_middleware` future flag in your `react-router.config.ts` file.
>
> ```ts {{ filename: 'react-router.config.ts' }}
> import type { Config } from '@react-router/dev/config'
>
> export default {
>   // ...
>   future: {
>     v8_middleware: true,
>   },
> } satisfies Config
> ```
