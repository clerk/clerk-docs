# Install the SDK

### Using Swift Package Manager (SPM)

Open the following menu item in Xcode:

File > Add Package Dependencies...

In the Search or Enter Package URL search box enter this URL:

```swift
https://github.com/clerk/clerk-ios
```

Then, select your preferred dependency rule and press Add Package.

<Callout>
For more information on SPM, visit the [official documentation](https://developer.apple.com/documentation/xcode/adding-package-dependencies-to-your-app").
</Callout>

### Configure and Load Clerk

<InjectKeys>

<Tabs type="framework" items={["SwiftUI", "UIKit"]}>

<Tab>
```swift filename="App.swift"
import SwiftUI
import ClerkSDK

@main
struct ClerkApp: App {
    var body: some Scene {
        WindowGroup {
            SignInView()
                .task {
                    Clerk.shared.configure(publishableKey: {{pub_key}})
                    try? await Clerk.shared.load()
                }
        }
    }
}
```
</Tab>

<Tab>
```swift filename="AppDelegate"
import UIKit
import ClerkSDK

func application(_ application: UIApplication, didFinishLaunchingWithOptions launchOptions: [UIApplication.LaunchOptionsKey: Any]?) -> Bool {
    Clerk.shared.configure(publishableKey: {{pub_key}})
    Task { try? await Clerk.shared.load() }
    return true
}
```
