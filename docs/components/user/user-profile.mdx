# `<UserProfile />`

<Images

The `<UserProfile/>` component is used to render a beautiful, full-featured account management UI that allows users to manage their profile and security settings.

## Usage


<CodeBlockTabs options={['Next.js', 'React', 'Remix']}>
```jsx copy filename="/pages/user-profile/[[...index]].tsx"
import { UserProfile } from "@clerk/nextjs";

const UserProfilePage = () => (
  <UserProfile path="/user-profile" routing="path" />
);

export default UserProfilePage;
```
```jsx copy filename="/user-profile.tsx"
import { UserProfile } from "@clerk/clerk-react";

const UserProfilePage = () => (
<UserProfile path="/user-profile" routing="path" />
);

export default UserProfilePage;
```
```tsx filename="routes/user/$.tsx" copy
import { UserProfile } from "@clerk/remix";

export default function UserProfilePage() {
  return <UserProfile path="/user" routing="path" />;
}
```
</CodeBlockTabs>
