The Backend `TestingToken` object holds information about a [Testing Token](/docs/guides/development/testing/overview#testing-tokens).

## Properties

| Property                           | Type     | Description                                |
| ---------------------------------- | -------- | ------------------------------------------ |
| <a id="expiresat"></a> `expiresAt` | `number` | The Unix timestamp when the token expires. |
| <a id="token"></a> `token`         | `string` | The token string.                          |
