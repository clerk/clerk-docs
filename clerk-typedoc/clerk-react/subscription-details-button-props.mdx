> **SubscriptionDetailsButtonProps** = <code>\{ for?: "organization" \| "user"; \}</code>

## Properties

| Property                | Type                                  | Description                                                                                                                                                                               |
| ----------------------- | ------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| <a id="for"></a> `for?` | <code>"organization" \| "user"</code> | The subscriber type to display the subscription details for. If `organization` is provided, the subscription details will be displayed for the active organization. Defaults to `'user'`. |
