The `OrganizationCreationDefaults` object holds the suggested default values to use when creating an Organization, along with an advisory surfacing a potential issue with the suggested defaults.

## Properties

| Property                         | Type                                                                                                                 | Description                                                                                                          |
| -------------------------------- | -------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------- |
| <a id="advisory"></a> `advisory` | <code>null \| \{ code: "organization_already_exists"; meta: Record\<string, string\>; severity: "warning"; \}</code> | An advisory surfacing a potential issue with the suggested defaults, or `null` if there is none.                     |
| `advisory.code`                  | `"organization_already_exists"`                                                                                      | The code identifying the advisory.                                                                                   |
| `advisory.meta`                  | `Record`\<`string`, `string`\>                                                                                       | Additional metadata providing context about the advisory.                                                            |
| `advisory.severity`              | `"warning"`                                                                                                          | The severity of the advisory.                                                                                        |
| <a id="form"></a> `form`         | <code>\{ blurHash: null \| string; logo: null \| string; name: string; slug: string; \}</code>                       | The suggested default values to pre-fill the Organization creation form with.                                        |
| `form.blurHash`                  | <code>null \| string</code>                                                                                          | The blur hash of the suggested logo, used to render a placeholder while the image loads, or `null` if there is none. |
| `form.logo`                      | <code>null \| string</code>                                                                                          | The suggested logo URL, or `null` if there is none.                                                                  |
| `form.name`                      | `string`                                                                                                             | The suggested Organization name.                                                                                     |
| `form.slug`                      | `string`                                                                                                             | The suggested URL-friendly identifier for the Organization.                                                          |
| <a id="id"></a> `id?`            | `string`                                                                                                             | The unique identifier of the resource.                                                                               |
