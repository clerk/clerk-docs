## Properties

| Property                                 | Type                                   | Description                                                                                                        |
| ---------------------------------------- | -------------------------------------- | ------------------------------------------------------------------------------------------------------------------ |
| <a id="primary"></a> `primary?`          | `boolean`                              | Indicates whether the Web3 Wallet is set as the primary Web3 Wallet, as multiple can be added to a user's profile. |
| <a id="strategy"></a> `strategy`         | `` `web3_${Web3Provider}_signature` `` | The strategy type.                                                                                                 |
| <a id="walletname"></a> `walletName?`    | `string`                               | The name of the Web3 Wallet.                                                                                       |
| <a id="web3walletid"></a> `web3WalletId` | `string`                               | The ID of the Web3 Wallet.                                                                                         |
