## Usage

<Include src="_partials/backend/usage" />

```tsx
const emailAddressId = 'idn_123'

const response = await clerkClient.emailAddresses.getEmailAddress(emailAddressId)
```

## Backend API (BAPI) endpoint

This method in the SDK is a wrapper around the BAPI endpoint `GET/email_addresses/{email_address_id}`. See the [BAPI reference](/docs/reference/backend-api/tag/email-addresses/GET/email_addresses/\{email_address_id}){{ target: '_blank' }} for more information.
