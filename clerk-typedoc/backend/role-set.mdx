The Backend `RoleSet` object represents a collection of roles that can be assigned to organization members.

## Properties

| Property                                         | Type                                                            | Description                                                                          |
| ------------------------------------------------ | --------------------------------------------------------------- | ------------------------------------------------------------------------------------ |
| <a id="createdat"></a> `createdAt`               | `number`                                                        | The date when the role set was first created.                                        |
| <a id="creatorrole"></a> `creatorRole`           | <code>null \| [RoleSetItem](role-set-item.mdx)</code>           | The role assigned to the creator of an organization.                                 |
| <a id="defaultrole"></a> `defaultRole`           | <code>null \| [RoleSetItem](role-set-item.mdx)</code>           | The default role assigned to new organization members.                               |
| <a id="description"></a> `description`           | <code>null \| string</code>                                     | A description of the role set.                                                       |
| <a id="id"></a> `id`                             | `string`                                                        | The unique identifier for the role set.                                              |
| <a id="key"></a> `key`                           | `string`                                                        | The unique key of the role set.                                                      |
| <a id="name"></a> `name`                         | `string`                                                        | The name of the role set.                                                            |
| <a id="roles"></a> `roles`                       | <code>[RoleSetItem](role-set-item.mdx)[]</code>                 | The roles that belong to the role set.                                               |
| <a id="rolesetmigration"></a> `roleSetMigration` | <code>null \| [RoleSetMigration](role-set-migration.mdx)</code> | Active migration information, only present when a migration is in progress.          |
| <a id="type"></a> `type`                         | <code>"initial" \| "custom"</code>                              | The type of the role set. `initial` role sets are the default for new organizations. |
| <a id="updatedat"></a> `updatedAt`               | `number`                                                        | The date when the role set was last updated.                                         |
