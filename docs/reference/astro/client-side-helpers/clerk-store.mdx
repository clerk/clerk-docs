The `$clerkStore` store provides a convenient way to access the [`Clerk`](/docs/reference/objects/clerk){{ target: '_blank' }} object. This provides access to some methods that are not available in other stores.

> [!WARNING]
> This is intended to be used for advanced use cases, like building a completely custom OAuth flow or as an escape hatch for getting access to the `Clerk` object.
