The `OrganizationMembershipRequest` object is the model that describes the request of a user to join an organization.

## Properties

| Property                         | Type     | Description                            |
| -------------------------------- | -------- | -------------------------------------- |
| <a id="id"></a> `id`             | `string` | The unique identifier of the resource. |
| <a id="pathroot"></a> `pathRoot` | `string` | The root path of the resource.         |
