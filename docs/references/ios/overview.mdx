# Clerk iOS SDK (Beta)

<Callout type="warning">
  The Clerk iOS SDK is currently in beta. It is **not yet recommended for production use**.
</Callout>

Clerk makes it simple to add authentication to your iOS application. This documentation covers the capabilities and methods available from Clerk's iOS SDK.

## SDK Reference

[SDK Reference](https://swiftpackageindex.com/clerk/clerk-ios/main/documentation/clerksdk)

## Installation

You can find the [Clerk iOS SDK](https://github.com/clerk/clerk-ios) on GitHub. Follow the instructions in the [readme](https://github.com/clerk/clerk-ios?tab=readme-ov-file#-installation) to install the Clerk iOS SDK using [Swift Package Manager](https://developer.apple.com/documentation/xcode/adding-package-dependencies-to-your-app).
