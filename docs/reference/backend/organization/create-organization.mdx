## Backend API (BAPI) endpoint

This method in the SDK is a wrapper around the BAPI endpoint `POST/organizations`. See the [BAPI reference](/docs/reference/backend-api/tag/organizations/POST/organizations){{ target: '_blank' }} for more information.
